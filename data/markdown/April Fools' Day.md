## Participating

Upon loading of game save, the april fools' day event will be active unless it was disabled beforehand. 

### Disabling the event

The event can be disabled using the "Disable Seasonal Events" option. This option is only available on the [title screen](/wiki/Game_menus "Game menus"), under [Options > Gameplay > Disable Seasonal Events](/wiki/Settings#Gameplay_2 "Settings"). 

In [multiplayer](/wiki/Multiplayer "Multiplayer"), the host's configuration is used. If the host has seasonal events enabled, but the client does not, the event will be enabled in the session. 

Alternatively, seasonal events can be disabled using the `-DisableSeasonalEvents` [launch argument](/wiki/Launch_arguments "Launch arguments"). On a [dedicated server](/wiki/Dedicated_server "Dedicated server"), this launch argument is the only way to disable seasonal events.

### [Fluffy-tailed Hog](/wiki/Fluffy-tailed_Hog "Fluffy-tailed Hog")

All Fluffy-tailed Hogs were changed to look like the [Walker](/wiki/Walker "Walker"), an enemy is from [Sanctum 2](https://sanctum.fandom.com/wiki/Sanctum_Wiki), another game by [Coffee Stain Studios](/wiki/Coffee_Stain_Studios "Coffee Stain Studios").

### [Cliff Hog and Nuclear Hog](/wiki/Cliff_Hog "Cliff Hog")

Starting with the April Fools Event 2024, all Cliff Hogs and Nuclear Hogs were changed to have a shape similar as the [Walker](/wiki/Walker "Walker"), but still used their default skin appearance.

### [Resource Scanner](/wiki/Resource_Scanner "Resource Scanner")

There were random capitalization of resource names in the Resource Scanner, except for [Coal](/wiki/Coal "Coal").

### [Flora](/wiki/Flora "Flora")
