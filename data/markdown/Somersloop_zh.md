# Somersloop/zh

## Somersloop  
  
[ ](/wiki/File:Somersloop.png "Somersloop.png")

_An odd alien object with a mind-bending, yet somehow familiar shape. It emits an unsettling energy._

### Class name

Desc_WAT1_C

## Item

### Stack size

50

**索莫晶体** 是在[世界](/wiki/World/zh "World/zh")各地发现的外星人工制品，用于[生产增幅](/wiki/Production_amplification/zh?action=edit&redlink=1 "Production amplification/zh \(page does not exist\)")和制造[](/wiki/Alien_Power_Augmenter "Alien Power Augmenter") [外星能源强化装置](/wiki/Alien_Power_Augmenter "Alien Power Augmenter")。 

它们会发出红光和不寻常的粒子，还会发出时大时小的脉动声。长时间站在它们旁边会听到声音，敦促玩家收集它们。

## 获取

在探索过程中可以发现索莫晶体，通常位于难以到达的地方，这些地方通常有敌人把守，周围有[铀矿藏](/wiki/Resource_Node/zh?action=edit&redlink=1 "Resource Node/zh \(page does not exist\)")、[孢子花](/wiki/Spore_Flower/zh?action=edit&redlink=1 "Spore Flower/zh \(page does not exist\)")、[破裂的巨石](/wiki/Cracked_boulder/zh?action=edit&redlink=1 "Cracked boulder/zh \(page does not exist\)")或[毒气柱](/wiki/Gas_Pillar/zh?action=edit&redlink=1 "Gas Pillar/zh \(page does not exist\)")。 [1.0版本](/wiki/Patch_1.0 "Patch 1.0")总共有106个索莫晶体。 

## 用法

### 生产增幅

_主要文章:_[生产增幅](/wiki/Production_amplifier "Production amplifier")

索莫晶体用于[生产增幅](/wiki/Production_amplifier/zh?action=edit&redlink=1 "Production amplifier/zh \(page does not exist\)")。在选定的生产建筑中插入索莫晶体，可以在不增加投入成本的情况下提高建筑的产出。最大提升幅度为 +100%，总产出翻倍。 

### 研究

[](/wiki/Somersloop/zh "Somersloop/zh") 索莫晶体在[](/wiki/MAM/zh "MAM/zh") [MAM研究站](/wiki/MAM/zh "MAM/zh")中被用来研究[外星科技研究链](/wiki/Alien_Technology/zh?action=edit&redlink=1 "Alien Technology/zh \(page does not exist\)")

## 琐事

  * 索莫晶体引用了“[Cool S](https://en.wikipedia.org/wiki/Cool_S "wikipedia:Cool S")”，这是一个流行的符号，通常被涂鸦在儿童笔记本上。它被描述为 “某种熟悉的形状”，这可能反映出 S 图的起源和传播并不明确。
