# High-Speed Connector

## High-Speed Connector

[ ](/wiki/File:High-Speed_Connector.png "High-Speed Connector.png")

_Connects multiple cables and wires in a very efficient way. Uses a standard pattern, so it has many varied applications._

### Unlocked by

[MAM Caterium Research](/wiki/Caterium_Research "Caterium Research") \- High-Speed Connector OR  
[Tier 7](/wiki/Tier_7 "Tier 7") \- Control System Development

### Class name

Desc_HighSpeedConnector_C

## Item

### Stack size

100

### [Sink points](/wiki/AWESOME_Sink "AWESOME Sink")

3,776

**High-Speed Connector** is a mid-game component used for advanced [power](/wiki/Power "Power") structures and other electronic items.
