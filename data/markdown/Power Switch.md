## Usage

Two separate Power Lines can be connected to the Power Switch, each one to either the 'A' connector on the left or 'B' connector on the right; the letter determines which graph displays which power grid. The switch only acts as a bridge; if there are any other connections joining the grids elsewhere, this switch will not separate them. Essentially, the Power Switch behaves the same as a [Power Line](/wiki/Power_Line "Power Line"). 

The connection is split by default and can be connected by pulling down the lever. The green light at the top left, the displayed words, and the 'chain' icon at the middle indicate the connection is linked. 

The [Indicator Light](/wiki/Indicator_Light "Indicator Light") on top of the Power Switch is green when the switch is turned on, yellow when it is turned off, and red if either one or both of the switch's connections are not connected to anything else. 

### Naming

Power Switches can be named by toggling the button at the top right in the UI. The name can be either typed in or pasted using [`Ctrl`](/wiki/Controls "Controls")+[`V`](/wiki/Controls "Controls") into the text box. There is no limitation on the name length, but only the first 20 characters will be displayed physically. Multiple Power Switches can share the same name. 

### Power trip

_Main article:_[ Power > Power trip](/wiki/Power#Power_trip "Power")

If the Power Switch is on and a non-tripped grid is linked with a tripped grid, the resulting combined grid will be tripped. Disabling the Power Switch may allow one grid to function while the other one remains tripped.
