This template is used to create tables for researches in the [MAM](/wiki/MAM "MAM"). 

## Parameters

Parameter | Description   
---|---  
name | Name of the research   
icon | Image to display as the icon (24px)   
quantityN | (N = 1-4) Amount of the required item   
costN | (N = 1-4) Name of the required item   
time | Time to research   
rewards | The rewards of the research   
tableEnd | Set to 1 on the last entry   
  
## Example
    
    
    {{ResearchTable
    |name = Green Power Slugs
    |icon = Green_Power_Slug.png
    |cost1 = Green Power Slug
    |quantity1 = 1
    |time = 00:03
    |rewards = Crafting Recipe: Green Power Slugs to [[Power Shard]]s
    }}
    {{ResearchTable
    |name = Yellow Power Slugs
    |icon = Yellow_Power_Slug.png
    |quantity1 = 1
    |cost1 = Yellow Power Slug
    |quantity2 = 25
    |cost2 = Reinforced Iron Plate
    |quantity3 = 10000
    |cost3 = Cable
    |time = 00:03
    |rewards = Crafting Recipe: Yellow Power Slugs to Power Shards
    |tableEnd = 1
    }}
