[ ](/wiki/File:Crafting_Bench.png "Crafting Bench.png")

The **Crafting Bench** is a manual crafting station, allowing to craft all unlocked components, except for: 

  * Equipment, which has to be crafted in an [Equipment Workshop](/wiki/Equipment_Workshop "Equipment Workshop")
  * [Alternate recipes](/wiki/Alternate_recipes "Alternate recipes"), [radioactive](/wiki/Radioactive "Radioactive") items and all [Space Elevator](/wiki/Space_Elevator "Space Elevator") project parts, all of which have to be made in machines
  * Recipes involving fluids, which have to be made in the [Refinery](/wiki/Refinery "Refinery"), [Packager](/wiki/Packager "Packager"), or [Blender](/wiki/Blender "Blender")
  * Recipes made in the [Particle Accelerator](/wiki/Particle_Accelerator "Particle Accelerator")



The [HUB](/wiki/HUB "HUB") has a Crafting Bench built into it, while additional Crafting Benches can be constructed elsewhere in the world using the [build menu](/wiki/Build_Gun "Build Gun"). Only one player can use a Crafting Bench in [multiplayer](/wiki/Multiplayer "Multiplayer"), so building multiple is advised.

## Usage

To craft a part, it has to be selected (or searched for) in the left side of the UI. If there are enough ingredients in the inventory, the Craft button turns yellow. The button can be pressed either by clicking on it [](/wiki/File:Keyboard_White_Mouse_Left.png "Left"), or held down automatically by tapping [`Space`](/wiki/Controls "Controls"), which is useful for crafting a large amount of items. 

After continuously crafting for a while, the hammer button will start glowing red, and boiling sounds can be heard. This has no effect on the game (any more[1]). 

Recipes are unlocked by completing [HUB Milestones](/wiki/Milestones "Milestones") and [MAM Researches](/wiki/MAM "MAM"). 

### Early game crafting speed comparison

Recipes in a Crafting Bench are marked with _hammer icon_ , which indicates how much time is required to produce one batch of items. A single hammer is equal to approximately 0.25 seconds (250 milliseconds). 

Let's consider an example. The [](/wiki/Iron_Ingot "Iron Ingot") [Iron Ingot](/wiki/Iron_Ingot "Iron Ingot") recipe has size of a batch that equals to 1. This means that 1 Iron Ingot is produced per recipe execution. Iron Ingot recipe has a cost of 3 hammers - that is, one Iron Ingot takes approximately 0.75 seconds (750 milliseconds) to be crafted. This results in production rate of 1.33 Iron Ingots per second. 

Below is the comparison of manual crafting speed vs machine crafting speed. This should give you some idea about which items should be prioritized for automation. Items with low manual crafting speed should be automated first. 'Machine' refers to either the [Constructor](/wiki/Constructor "Constructor") or the [Smelter](/wiki/Smelter "Smelter"), which both operate at 4 MW.
