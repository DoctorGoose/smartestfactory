# Template:See

[](/wiki/File:Template-info.svg) Documentation  
  
[[purge](https://satisfactory.wiki.gg/wiki/Template:See?action=purge)]

This template is used after the heading of the summary, to link to the sub-article that has been (or will be) summarised. It uses `{{[hatnote](/wiki/Template:Hatnote "Template:Hatnote")}}`. 

## Usage

`{{See|Article 1}}`

You may also include additional links, up to 5 in total: 

`{{See|Article 1|Article 2|...|Article 5}}`
