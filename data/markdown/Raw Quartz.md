# Raw Quartz

## Raw Quartz

[ ](/wiki/File:Raw_Quartz.png "Raw Quartz.png")

_Can be processed into Quartz Crystals and Silica, which both offer a variety of applications._

### Unlocked by

[Tier 9](/wiki/Tier_9 "Tier 9") \- Matter Conversion

### Class name

Desc_RawQuartz_C

## Item

### Stack size

100

**Raw Quartz** is an uncommon mid-game resource found in the [world](/wiki/World "World"). It is processed into [Quartz Crystals](/wiki/Quartz_Crystal "Quartz Crystal") or [Silica](/wiki/Silica "Silica") for further processing.

### AWESOME Sink

**[](/wiki/Raw_Quartz "Raw Quartz")Raw Quartz** can be sunk into the [](/wiki/AWESOME_Sink "AWESOME Sink") [AWESOME Sink](/wiki/AWESOME_Sink "AWESOME Sink") to yield **15 points**.

## Tips

  * Raw Quartz is a great source of [AWESOME Sink](/wiki/AWESOME_Sink "AWESOME Sink") points in the early game given how early it can be obtained and the number of points it gives before it becomes useful in later tiers.
  * If being mined for the purpose of early [AWESOME Sink](/wiki/AWESOME_Sink "AWESOME Sink") point farming, processing Raw Quartz into [Quartz Crystals](/wiki/Quartz_Crystal "Quartz Crystal") or [Silica](/wiki/Silica "Silica") before sinking it is a viable option to consider, if you are willing to build some [Constructors](/wiki/Constructor "Constructor") and can afford the additional [power](/wiki/Power "Power") requirement. Of the two options, [Silica](/wiki/Silica "Silica") will improve the point yield slightly more than [Quartz Crystals](/wiki/Quartz_Crystal "Quartz Crystal"), but will require more total Constructors to process the same amount of Raw Quartz.
