# Patch 0.5.0.10

Patch Notes: Early Access (EXPERIMENTAL) – **v0.5.0.10** – Build 172929. This patch was released on 16 November 2021.   
  
The original post can be viewed on Satisfactory's [Discord](https://discord.com/channels/370472939054956546/557603412565426196/910252265561022526), [Reddit](https://www.reddit.com/r/SatisfactoryGame/comments/qvg32c), and [Steam](https://store.steampowered.com/news/app/526870/view/4316132205203201679). 

## Contents

  * 1 Intro
  * 2 Renderer
  * 3 Bug fixes
  * 4 Quality of Life
  * 5 Localisation



## Intro

Hi Pioneers! 

Hello everyone, today’s patch should finally fix some of the issues that we’ve fixed a few times but keep coming back, as they say, fifth time’s a charm, if they don’t work this time, they are haunted or something I dunno. (We’ll try yet again to fix them of course) 

As always, if we’ve made things better or way worse, please let us know over at our QA site <https://questions.satisfactorygame.com/> We read your comments every day 

See you all again soon! <3 

## Renderer

  * Made DX12 the [default graphics renderer](/wiki/Unreal_Engine "Unreal Engine"). When not supported the game should still default to DX11. [Options Menu](/wiki/Settings#Video "Settings") might show DX12 as selected even though DX11 is the one active in these cases. (Known Bug)



## Bug fixes

  * Fixed incorrect default materials on some [Foundations](/wiki/Foundations "Foundations")/[Walls](/wiki/Walls "Walls") after the introduction of the “Concrete Foundation”, “Grip Metal Foundation” and “Coated Concrete Foundation” materials introduced last patch
  * Fixed Options Menu not scrolling down all the way in certain menus
  * Fixed [Fuel Generator](/wiki/Fuel_Generator "Fuel Generator") [Status Indicator](/wiki/Indicator_Light "Indicator Light") being offset
  * Fixed “Current Objective” display in [HUD](/wiki/HUD "HUD") showing the wrong requirements for some objectives
  * Fixed visual bug where [Conveyor Lifts](/wiki/Conveyor_Lifts "Conveyor Lifts") would extend upwards when snapping from a [Floor Hole](/wiki/Conveyor_Poles#Floor-0 "Conveyor Poles")
  * Fixed more [Beam](/wiki/Beams "Beams") Rotation issues
  * Fixed Beam rotation issues when snapping to Supports and Connectors
  * Made most [bushes](/wiki/Flora "Flora") destructible by [Nobelisks](/wiki/Nobelisk "Nobelisk")
  * Made some small flowers in [Northern Forest](/wiki/Northern_Forest "Northern Forest") interactable again
  * Fixed a bunch of visual issues with Conveyor Lift snapping
  * Fixed bug where Beam dismantle returns wouldn’t match the build cost
  * Fixed the search function in the [Customizer](/wiki/Customizer "Customizer") not working properly



## Quality of Life

  * Added Lizard Doggo to [Sign](/wiki/Signs "Signs") Icons (Icon > Image Browser > Other)



## Localisation

  * Updated all languages with the latest translations and published new content for translation
  * Updated language completion rates in the language menu selection
