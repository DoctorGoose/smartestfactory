This template creates a header with an optional image overlaid to the right or left. Used on the front page within fpboxes. It uses the images in [Category:Header part images](/wiki/Category:Header_part_images "Category:Header part images"). 

## Usage

`{{[header](/wiki/Template:Header "Template:Header")|header text|[[File:example.png]]|additional classes}}`
