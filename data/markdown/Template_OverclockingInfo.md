Creates descriptions of overclocking based on the value of `{{{1|}}}`. 

## Generator

`{{OverclockingInfo|generator}}` produces:  


_Main article:_[Clock speed](/wiki/Clock_speed "Clock speed")

The OverclockingInfo can be overclocked using [Power Shards](/wiki/Power_Shard "Power Shard"). Overclocking increases the fuel (and Water) consumption rate and produced power linearly. The same amount of power is always produced from the same input resources. 

Optional parameter `fuel`, which replaces "fuel" in "As the _fuel_ consumption rate is directly proportional".  
`{{OverclockingInfo|generator|fuel=Coal and Water}}`

## Consumer

`{{OverclockingInfo|consumer}}` produces:  


_Main article:_[Clock speed](/wiki/Clock_speed "Clock speed")

The OverclockingInfo can be overclocked using [Power Shards](/wiki/Power_Shard "Power Shard"). Overclocking increases the input/output speed of the OverclockingInfo at the cost of exponentially increased [power](/wiki/Power "Power") demand. 

Expects a `power` parameter, else fallbacks to 1  
`{{OverclockingInfo|consumer|power=4}}`

## Default

`{{OverclockingInfo}}` produces:  


_Main article:_[Clock speed](/wiki/Clock_speed "Clock speed")

The OverclockingInfo cannot be overclocked.
