# Patch 0.5.0.4

Patch Notes: Early Access (EXPERIMENTAL) – **v0.5.0.4** – Build 170618. This patch was released on 28 October 2021.   
  
The original post can be viewed on Satisfactory's [Discord](https://discord.com/channels/370472939054956546/557603412565426196/903375216657436683), [Reddit](https://www.reddit.com/r/SatisfactoryGame/comments/qhvgqe), and [Steam](https://store.steampowered.com/news/app/526870/view/3095655347292437150).

## Intro

Hi Pioneers! 

Hello again everybody, here’s another patch with a bunch of fixes, including some of the things we’ve seen reported the most in our QA site and other places 

As always, if we introduced new issues or if we haven’t fixed something that you think we should, please let us know over at our QA Site: <https://questions.satisfactorygame.com/> We read your posts every day 

We hope you all have a great weekend and enjoy playing our game, thanks for all your support <3 

## Bug fixes

  * Fixed up [clearance snapping](/wiki/Buildings "Buildings") rotation issues
  * Fixed a bunch of UI issues that led to Clients not being able to set Timetables in [Multiplayer](/wiki/Multiplayer "Multiplayer")
  * Fixed [Hyper Tube Entrances](/wiki/Hypertube_Entrance "Hypertube Entrance") not properly snapping to [Hyper Tube Supports](/wiki/Hypertube_Supports#Simple-0 "Hypertube Supports")
  * Added some extra safety checks for [dismantling](/wiki/Build_Gun "Build Gun") to hopefully prevent some crashes
  * Re-added missing [Bauxite](/wiki/Bauxite "Bauxite") [nodes](/wiki/Resource_Node "Resource Node")
  * Re-added missing [Uranium](/wiki/Uranium "Uranium") [Resource Node](/wiki/Resource_Node "Resource Node")
  * Fixed [`G`](/wiki/Controls "Controls") key being unable to be rebound
  * Fixed [Pipeline](/wiki/Pipeline_Supports#Wall-0 "Pipeline Supports") and [Hyper Tube Wall](/wiki/Hypertube_Supports#Wall-0 "Hypertube Supports") attachments being offset
  * Fixed [Beam](/wiki/Beams "Beams") rotation when building them vertically
  * Potential fix for a Crash related to [Instance Bucket](/wiki/Unreal_Engine "Unreal Engine")
  * Fix for the [Train](/wiki/Train "Train") payload being reset on load and staying like that until the next time the train is loaded
  * Fixed [Dedicated Servers](/wiki/Dedicated_Servers "Dedicated Servers") crash related to [Vehicles](/wiki/Vehicles "Vehicles") end play
  * Fixed Client for crash related to not-yet-replicated member
  * Temporary fix for Trains getting stuck in Docking State, Docking animations might not work until a proper fix is applied
  * Fixed [Pipe Fluid indicators](/wiki/Pipelines#Flow_Indicator "Pipelines") not showing the correct fluid
  * Fixed crash related to the Hotbar when Client joins a game in Multiplayer
  * Fixed a crash when joining a game via invites or online status on Steam
  * Fixed some cases where invites sent from Steam weren’t being delivered
  * Fixed a crash when using [Parachutes](/wiki/Parachute "Parachute") as Client
  * Fixed unlocked visible areas in the [Map](/wiki/Map "Map") getting reset
  * Fixed Pipes displaying Headlift, now they display Flow Rate (Pumps display both)
  * Corrected the [Corner Ramp](/wiki/Foundations "Foundations") rotation (This means that if you’ve manually corrected them yourself, you will have to rebuild them, sorry for the inconvenience)



## Quality of Life

  * [Conveyor](/wiki/Conveyor_Poles#Types "Conveyor Poles") / [Pipeline](/wiki/Pipeline_Supports#Types "Pipeline Supports") Stackable poles changed to have soft clearance
  * Added zooping to Quarter Pipe foundations
  * [Conveyor Lifts](/wiki/Conveyor_Lifts "Conveyor Lifts") partially changed to have soft clearance
  * Floor holes partially changed to have soft clearance
  * Changed so the snap guidelines always appear (The Green line to align to inputs/outputs)
  * You can now rebind the [`Left Alt`](/wiki/Controls "Controls") key for Quick Apply Material in Hotbar
  * [Shopping List](/wiki/To-Do_List "To-Do List") colours are now clearer to distinguish



## Dedicated server

  * Changed "-GamePort" server parameter to "-Port"
  * Added a “Save Game” server command
  * Flipped around the Show/Hide Address Toggle Button text in the Add Server popup



## Localisation

  * Updated the new entries in the Options Menu to make them localisable
