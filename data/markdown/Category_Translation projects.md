# Category:Translation projects

These are the **community-maintained translation projects** of wiki articles into languages other than English. 

Translation projects adhere to the [Localization project rules](/wiki/Satisfactory_Wiki:Localization_project_rules "Satisfactory Wiki:Localization project rules"). If you would like to help translating or start a new translation project, please ask on the [Admin noticeboard](/wiki/Satisfactory_Wiki:Admin_noticeboard "Satisfactory Wiki:Admin noticeboard").
