**Biomass** is a fuel item produced from various forms of biomass items such as [](/wiki/Leaves "Leaves") [Leaves](/wiki/Leaves "Leaves"), [](/wiki/Wood "Wood") [Wood](/wiki/Wood "Wood"), [](/wiki/Mycelia "Mycelia") [Mycelia](/wiki/Mycelia "Mycelia") or [](/wiki/Alien_Protein "Alien Protein") [Alien Protein](/wiki/Alien_Protein "Alien Protein"). It is used to fuel the [Biomass Burner](/wiki/Biomass_Burner "Biomass Burner") and [vehicles](/wiki/Vehicle "Vehicle"), providing significantly more energy than [](/wiki/Leaves "Leaves") [Leaves](/wiki/Leaves "Leaves") but less than [](/wiki/Solid_Biofuel "Solid Biofuel") [Solid Biofuel](/wiki/Solid_Biofuel "Solid Biofuel"). It is also the building material for [U-Jelly Landing Pads](/wiki/U-Jelly_Landing_Pad "U-Jelly Landing Pad"). 

For conversion rates, see [Biomass Burner § Conversion](/wiki/Biomass_Burner#Conversion "Biomass Burner").

## Trivia

  * Over-harvesting of foliage in the game has an impact on game performance, more noticeably for clients in [multiplayer](/wiki/Multiplayer "Multiplayer"). This includes any trees chopped down, any flower or bushes picked up, including small rocks and stones that can be destroyed. The game stores the complete map including all trees and others as the "default" state. For every single entity removed from the environment, the game has to save the additional information for that removal. 
    * For example, if 100 trees are chopped down, then the map state will be stated as "Default map with all trees + tree 1 chopped down + tree 2 chopped down + ... + tree 100 chopped down".
    * On the client side, ghost trees can often be seen, because the default map is loaded first, and the data with tree removal can take some time to load. This can lead to desyncs between each player.
