# Cable

## Cable

[ ](/wiki/File:Cable.png "Cable.png")

_Used for crafting.  
Primarily used to build Power Lines._

### Unlocked by

[Tier 0](/wiki/Tier_0 "Tier 0") \- HUB Upgrade 2

### Class name

Desc_Cable_C

## Item

### Stack size

200

### [Sink points](/wiki/AWESOME_Sink "AWESOME Sink")

24

**Cables** are a basic component made from [Wires](/wiki/Wire "Wire"). They are used for building [Power Lines](/wiki/Power_Line "Power Line"), several parts, and many other buildings.
