The **Chainsaw** is a tool used to cut down [flora](/wiki/Category:Flora "Category:Flora") in a target area by consuming [](/wiki/Solid_Biofuel "Solid Biofuel") [Solid Biofuel](/wiki/Solid_Biofuel "Solid Biofuel"). It allows the [pioneer](/wiki/Pioneer "Pioneer") to harvest things that can't be harvested by hand, such as [trees](/wiki/Trees "Trees"), some [shrubs](/wiki/Shrub "Shrub"), and small rocks, or to clear an area of these things.

## Usage

### Equipment

With the Chainsaw equipped, approach some flora (valid flora will be highlighted with a white outline) and Use [](/wiki/File:Keyboard_White_Mouse_Left.png "Left") to cut it. After the animation finishes, all flora within a 5-meter radius is cut (centered on the targeted object).[1] All items produced by the destroyed flora will be transferred into the player's inventory. Several types of trees cannot be cut, usually large trees. 

The Chainsaw can be toggled into Single Mode with the [`R`](/wiki/Controls "Controls") key, in which mode it will cut down only the targeted object. 

### Damage

The Chainsaw cannot be used as a weapon; it does no [damage](/wiki/Damage "Damage") to any creature. 

### Crafting

The Chainsaw is primarily used to collect [Leaves](/wiki/Leaves "Leaves"), [Wood](/wiki/Wood "Wood"), and [Mycelia](/wiki/Mycelia "Mycelia") ingredients, which are all used to create [Biomass](/wiki/Biomass "Biomass"). 

### AWESOME Sink

**[](/wiki/Chainsaw "Chainsaw")Chainsaw** can be sunk into the [](/wiki/AWESOME_Sink "AWESOME Sink") [AWESOME Sink](/wiki/AWESOME_Sink "AWESOME Sink") to yield **2,760 points**.

### Fuel consumption

The Chainsaw uses [Solid Biofuel](/wiki/Solid_Biofuel "Solid Biofuel") as fuel. It consumes 75 MW and Solid Biofuel has 450 MJ of energy, therefore one Solid Biofuel is consumed every six seconds. 

While changing the fuel type is a supported UI feature for [Jetpacks](/wiki/Jetpack "Jetpack"), there isn't currently a way for the player to change which fuel the Chainsaw should consume.[2]

## Trivia

  * In Closed Alpha the Chainsaw Graphic Mesh used a Blue Color. 
    * In Closed Alpha, when held in one hand, the Chainsaw Blade was retracted.[3]
    * In Closed Alpha, when held in two hands, the Chainsaw Blade was extended.[4]
  * Despite being powered by [a solid form](/wiki/Solid_Biofuel "Solid Biofuel") of [biomass](/wiki/Biomass "Biomass"), the chainsaw still sounds like it is powered by liquid fuel.[5]
  * Snutt has prototyped different modes for the Chainsaw.[6]
  * Even if Chainsaw is equipped, Pioneer can still opt to pick up small foliages manually. However, as foliage actual pickup hitbox is reduced to its base area while a Chainsaw is held, Holster [`H`](/wiki/Controls "Controls") to allow foliages to be picked up more easily.
  * [Beryl Nut](/wiki/Beryl_Nut "Beryl Nut") bushes and [Paleberry](/wiki/Paleberry "Paleberry") plants within the Chainsaw's cutting radius will not be destroyed unless targeted with the Chainsaw directly. When targeted with the Chainsaw, any uncollected Beryl Nuts or Paleberries will be harvested, and the plants will drop no other items.
  * The equip animation that plays the first time the player equips the chainsaw is a reference to the chainsaw in DOOM (2016).
