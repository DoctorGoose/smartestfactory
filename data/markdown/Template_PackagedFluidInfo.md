# Template:PackagedFluidInfo

**PackagedFluidInfo** is the packaged form of [PackagedFluidInfo](/wiki/PackagedFluidInfo?action=edit&redlink=1 "PackagedFluidInfo \(page does not exist\)") that can be carried in inventories and transported via [Conveyor Belts](/wiki/Conveyor_Belt "Conveyor Belt") instead of [Pipelines](/wiki/Pipeline "Pipeline"). It is made in [Packagers](/wiki/Packager "Packager") using [](/wiki/Empty_Canister "Empty Canister") [Empty Canisters](/wiki/Empty_Canister "Empty Canister"), where it can be also unpackaged, refunding the used Canisters. 

[](/wiki/File:Template-info.svg) Documentation

[[purge](https://satisfactory.wiki.gg/wiki/Template:PackagedFluidInfo?action=purge)]

Used for a unified description of all packaged fluids. It has one optional parameter, the name of the packaged item, which defaults to the page name. The template automatically removes "Packaged " from the name in the description.
