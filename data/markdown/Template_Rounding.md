# Template:Rounding

This template rounds a number yet displays the original precision when mouse hovered. Example:   
`{{Rounding|0.123456|2}}`  
0.12   
`{{Rounding|4321.123456|4}}`  
4321.1235   
(observe the last digit has been rounded up as the truncated digit is 5 and above)
