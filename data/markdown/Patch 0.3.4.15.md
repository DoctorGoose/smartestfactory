# Patch 0.3.4.15

Patch Notes: Early Access – **v0.3.4.15** – Build 123924. This patch was released on 4 June 2020.   
  
The original post can be viewed on Satisfactory's [Discord](https://discord.com/channels/370472939054956546/557603412565426196/718124636726100010) and [Reddit](https://www.reddit.com/r/SatisfactoryGame/comments/gwjo66). 

## Contents

  * 1 Intro
  * 2 UI
  * 3 Localisation
  * 4 Factory
  * 5 Bug fixes



## Intro

Hi Pioneers! 

We are pushing a patch, mainly to address the current issue that Extractors stop functioning in multiplayer. That should be resolved now. There are several other bug fixes and smaller UI tweaks included, that you can check out in the patch notes below. 

We also updated the building clearance of the Empty Train Platform, so other buildables can be built in the empty space above it, and we updated all translation files again and included Hebrew translations and font support. 

Hope you all have a good time with this patch and can enjoy multiplayer again <3 

## UI

  * Updated the in-game [Map](/wiki/Map "Map") with latest changes
  * Small tweaks to the Join Game menus player icons and information display
  * Restyled social media buttons in the Main menu
  * Grouped login and NAT type information together in the Main menu
  * [MAM](/wiki/MAM "MAM") button now has text wrapping



## Localisation

  * Added Hebrew translations and font support
  * Updates all translation files



## Factory

  * Reduce the building clearance size of the [Empty Train Platform](/wiki/Empty_Platform "Empty Platform")



## Bug fixes

  * Extractors should continue working as usual after a client player opens their menu
  * Sub-buildings (like the Train Track on Platforms or the [Craft Bench](/wiki/Craft_Bench "Craft Bench") in [the Hub](/wiki/The_HUB "The HUB")) should no longer rotate separately from the main building
  * Fixed some [Uranium](/wiki/Uranium "Uranium") and [SAM](/wiki/SAM_Ore "SAM Ore") [nodes](/wiki/Resource_Node "Resource Node") that had the [Iron](/wiki/Iron_Ore "Iron Ore") material on them instead of their own
  * Newly built [Tracks](/wiki/Railway "Railway") are no longer invisible for Client players, but they currently don’t show their build effect anymore
  * Fixed a memory leak and crash related to building [Conveyor Belts](/wiki/Conveyor_Belts "Conveyor Belts"), [Pipes](/wiki/Pipe "Pipe"), Train Tracks & [Hyper Tubes](/wiki/Hyper_Tube "Hyper Tube")
  * [Crash Sites](/wiki/Crash_Site "Crash Site") in the Red Jungle now have rewards
  * [Fuel Generators](/wiki/Fuel_Generator "Fuel Generator") are fully paintable with the [Color Gun](/wiki/Color_Gun "Color Gun") again
