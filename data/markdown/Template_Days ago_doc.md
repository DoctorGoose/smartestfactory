This template makes use of [ParserFunctions](http://www.mediawiki.org/wiki/Help:Extension:ParserFunctions) in the MediaWiki software to calculate the amount of days past since a specified date. Its main usage is on the [Main Page](/wiki/Main_Page "Main Page") to display how long the latest patch, test patch, blog post and version has been released. 

## Usage

`{{[Days ago](/wiki/Template:Days_ago "Template:Days ago")}}`

## Example

  * `{{[Days ago](/wiki/Template:Days_ago "Template:Days ago")}}` generates 5442 days ago
