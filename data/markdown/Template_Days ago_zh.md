# Template:Days ago/zh

[](/wiki/File:Template-info.svg) Documentation

[[purge](https://satisfactory.wiki.gg/wiki/Template:Days_ago/zh?action=purge)]

这个模板是用 MediaWiki software 里的 [ParserFunctions](http://www.mediawiki.org/wiki/Help:Extension:ParserFunctions) 来计算指定日期之后的天数。 它的主要用途是在 [主页](/wiki/Main_Page "Main Page") 来显示最新补丁、测试补丁、发布的文章和版本的发布时间。 

## 用法

`{{[Days ago](/wiki/Template:Days_ago "Template:Days ago")}}`

## 例子

  * `{{[Days ago](/wiki/Template:Days_ago "Template:Days ago")}}` generates 5428 days ago
