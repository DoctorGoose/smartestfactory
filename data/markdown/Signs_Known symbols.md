### Other Formatting

In addition to using Copy/Paste, a number of special characters can be entered using their windows code page numbers[2]. Alt codes for the common Windows-1252 code page[3][4] used for most English and European locales are listed below. In order to enter a character by it's code page number, ensure that NumLock is on, then hold down the Alt key and enter 0, followed by the code page number on the numeric keypad while still holding Alt.
