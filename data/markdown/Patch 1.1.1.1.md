# Patch 1.1.1.1

Patch Notes: **v1.1.1.1** – Build 418783. This patch was released on 18 June 2025.   
  
The original post can be viewed on Satisfactory's [Discord](https://discord.com/channels/370472939054956546/557603412565426196/1384922455243620453), [Reddit](https://www.reddit.com/r/SatisfactoryGame/comments/1lekhyj), and [Steam](https://store.steampowered.com/news/app/526870/view/520842750961123376). 

## Contents

  * 2 Bug fixes
  * 3 Controller
  * 4 Localisation



## Intro

Hi Pioneers! 

Hello again everyone, We have a patch today for you today with some fixes to alleviate issues some of you are having before the weekend 

We’re still working on some other issues that we have identified which will come in on a later patch, but for now If there’s any more major issues you’re having, please let us know over at our QA Site <https://questions.satisfactorygame.com/> All your posts are super helpful 

See you all again and thank you for the warm reception for 1.1, hope you’re all enjoying it <3 

## Bug fixes

  * Potential fix for [Dedicated Server](/wiki/Dedicated_Server "Dedicated Server") crash when unequipping the [Zipline](/wiki/Zipline "Zipline")
  * Fix for [Conveyor Belts](/wiki/Conveyor_Belts "Conveyor Belts") built using [Blueprint](/wiki/Blueprint "Blueprint") Auto Connections not working after saving and reloading 
    * Existing Conveyor Belts that are affected by this issue will have to be rebuilt to work properly after this fix
  * Fixed Blueprint Auto Connect sometimes flipping the direction of the hologram when building a Blueprint that contained [Railways](/wiki/Railway "Railway")
  * Fixed [Welcome Screen](/wiki/Main_menu "Main menu") showing up on every launch on Steam
  * Potential fix for Crash in [Multiplayer](/wiki/Multiplayer "Multiplayer") related to Conveyor Chain Replication Data



## [Controller](/wiki/Controller "Controller")

  * Fixed UI button hints for the [Valve](/wiki/Valve "Valve")
  * Adjusted behavior when adjusting the Valve Limit with the DPAD
  * Fixed Focus loss issue when changing the [Session Settings/Session Type](/wiki/Multiplayer#Hosting_a_session "Multiplayer") while disconnected from EOS
  * Fixed Welcome Screen not being interactable in some scenarios for [Steam Deck](/wiki/Controller#Steam_Deck "Controller")



## Localisation

  * Updated community translated languages with the latest translations
  * Updated language completion rates
  * Fixes for multiple missing localization bugs
