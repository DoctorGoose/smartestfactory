# Template:Wiki admin

[](/wiki/File:Template-info.svg) Documentation

[[purge](https://satisfactory.wiki.gg/wiki/Template:Wiki_admin?action=purge)]

This is a simple template providing standardized formatting for entries in [Template:Admins](/wiki/Template:Admins "Template:Admins"). 

**Usage** : 

    `{{Wiki admin|account name|decoration (optional)}}`

`decoration` is optional, and will be placed after the username but before the talk and contribution links. Intended for use with language flags or titles. 

**Note** : The template must be enclosed within a table and does not include a row break character. 

**Example** : 

    `{{Wiki admin|Game widow|(Wiki Team Lead) [[File:Flag qc.svg|16px|Français|link=]]}}`

**Result** : 

[Game widow](/wiki/User:Game_widow "User:Game widow") (Wiki Team Lead)  | [_talk_](/wiki/User_talk:Game_widow "User talk:Game widow") • [_contribs_](/wiki/Special:Contributions/Game_widow "Special:Contributions/Game widow")  
---|---
