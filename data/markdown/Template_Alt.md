# Template:Alt

A small text that indicates a recipe to be an alternative version to the default recipe. These can be unlocked by scanning [Hard Drives](/wiki/Hard_Drive "Hard Drive"), so this snippet will link to that page. 

## Example

`{{Alt}}`

[Alt](/wiki/Hard_Drive "Hard Drive")
  *[v]: View this template
  *[e]: Edit this template
  *[h]: History of this template
  *[his]: his trivia section
