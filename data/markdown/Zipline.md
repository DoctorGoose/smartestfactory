[ ](/wiki/File:Zipline.png "Zipline.png")

### Stack size

### [Sink points](/wiki/AWESOME_Sink "AWESOME Sink")

The **Zipline** is an early-game piece of hand [equipment](/wiki/Equipment "Equipment") that can be used for navigating along [Power Lines](/wiki/Power_Line "Power Line"). Despite statements in pre-release teasers, it is not necessary for the Power Line to be powered, meaning it can be effectively used for exploration or crossing gaps.

## Obtaining

The Zipline tool is the first item in the game not to be "official FICSIT-sanctioned equipment",[1] having been jury-rigged by the [Pioneer](/wiki/Pioneer "Pioneer") in their off time from existing parts & technology such as the "newly-discovered [Caterium](/wiki/Caterium "Caterium")",[1] and the FICSIT-branded adhesive tape from the [Nobelisk](/wiki/Nobelisk "Nobelisk").[2]

## Usage

### Equipment

While equipping a Zipline in the hand slot, approaching a [Power Line](/wiki/Power_Line "Power Line") and holding [](/wiki/File:Keyboard_White_Mouse_Left.png "Left") or pressing [](/wiki/File:Keyboard_White_Mouse_Right.png "Right") to toggle on/off uses the Zipline to travel along the Power Line. Once attached to the Zipline, the player will move forward in the direction the player is looking. 

Ziplining can be stopped by letting go of [](/wiki/File:Keyboard_White_Mouse_Left.png "Left") or pressing [](/wiki/File:Keyboard_White_Mouse_Right.png "Right"). Ziplining will also end if the player bumps into anything. 

It is possible to continue travelling between multiple Power Lines connected at a [Wall Outlet](/wiki/Wall_Outlet "Wall Outlet") or [Power Tower](/wiki/Power_Tower "Power Tower"). [Power Tower Platforms](/wiki/Power_Tower_Platform "Power Tower Platform") are specifically designed for this purpose. The desired Power Line can be selected by looking in its direction, unless it exceeds the maximum turn angle of 60 degrees. 

Ziplining on a level Power Line is just as fast as running with Blade Runners, 13.5 m/s. Traveling down a diagonal Power Line will speed up travel and vice versa, zipping up diagonal / almost vertical Power Lines will slow down to ~5 m/s. It is not possible to zip along vertical Power Lines. 

Sprinting while ziplining speeds up the movement. The sprint key ([`Left-shift`](/wiki/Controls "Controls") has to be pressed each time the game is restarted, and will be remembered if the [Hold to sprint option](/wiki/Settings#Controls "Settings") is disabled (OFF).

Up to 500 meters have been tested.[_[citation needed](/wiki/Category:Citation_needed "Category:Citation needed")_]

## History

  * [Patch 1.1.1.1](/wiki/Patch_1.1.1.1 "Patch 1.1.1.1"): Potential fix for [Dedicated Server](/wiki/Dedicated_Server "Dedicated Server") crash when unequipping the Zipline
  * [Patch 1.0](/wiki/Patch_1.0 "Patch 1.0")
    * Changed particle effects while ziplining
  * [Patch 0.8.2.0](/wiki/Patch_0.8.2.0 "Patch 0.8.2.0")
    * Increased Zipline sprint acceleration. It should be easier to tell that sprint mode is turned on now due to the more direct change in speed
    * Zipline path selection now works vertically too
    * Additional improvements to try and mitigate cases where you get disconnected from ziplining when ziplining on an incline
    * Fixed Jumping off a zipline canceling the Zipline input if it's still held, now holding it will continue the input as it should
  * [Patch 0.8.0.4](/wiki/Patch_0.8.0.4 "Patch 0.8.0.4"): Reduced the height at which the player hangs from the Zipline, this should help some cases where the player gets detached from the cable when ziplining down an incline due to bumping the Pioneer’s head
  * [Patch 0.8.0.0](/wiki/Patch_0.8.0.0 "Patch 0.8.0.0"): 
    * The Zipline now automatically connects to the next [Power Line](/wiki/Power_Line "Power Line") while ziplining on [Power Towers](/wiki/Power_Tower "Power Tower") and [ceiling connections](/wiki/Wall_Outlet "Wall Outlet")
      * This will not work if the angle between the Power Lines exceeds 60 degrees
      * If the Zipline can move along more than one Power Line the direction can be controlled by looking at the desired direction or by holding the directional movement input
    * Reduced maximum Zipline Sprint Speed to 100 km/h (due to the previous maximum speed being too high for the world loading)
    * Increased the Zipline Sprint acceleration to be nearly instant
    * The Zipline can now be toggled on/off by pressing the secondary equipment input (RMB by default), which allows ziplining without needing to hold down the primary equipment input (LMB by default)
  * [Patch 0.7.0.8](/wiki/Patch_0.7.0.8 "Patch 0.7.0.8"): Added sprint function
  * [Patch 0.7.0.4](/wiki/Patch_0.7.0.4 "Patch 0.7.0.4"): Fixed Zipline sprinting not working for Clients
  * [Patch 0.7.0.0](/wiki/Patch_0.7.0.0 "Patch 0.7.0.0"): Added sprinting functionality for the Zipline
  * [Patch 0.4.0.5](/wiki/Patch_0.4.0.5 "Patch 0.4.0.5"): Fixed the Zipline issues introduced in the last patch
  * [Patch 0.4.0.4](/wiki/Patch_0.4.0.4 "Patch 0.4.0.4"): 
    * Reworked correction velocity for the Zipline which should fix players getting launched in the air when using it
    * Zipline now conserves velocity when you start ziplining
  * [Patch 0.4.0.3](/wiki/Patch_0.4.0.3 "Patch 0.4.0.3"): 
    * Fixed rare crash which could happen when using the Zipline
    * Fixed an extremely rare crash which could happen when trying to drop from a Zipline by crouching
    * Increased Zipline speed and decreased speed reduction when zipping downwards
  * [Patch 0.4.0.0](/wiki/Patch_0.4.0.0 "Patch 0.4.0.0"): Introduced
