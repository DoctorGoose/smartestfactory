# Template:Shopunlock

[](/wiki/File:Template-info.svg) Documentation

[[purge](https://satisfactory.wiki.gg/wiki/Template:Shopunlock?action=purge)]

The Shopunlock template is used in the Obtaining section of buildings under Unlocking. It displays that the building is unlocked in the AWESOME Shop using a certain amount of FICSIT Coupons, and optionally with and additional unlock requirement. 

## Usage

### Parameters

Parameters have to be entered in order. 

Parameter | Data type | Description   
---|---|---  
1 | Integer | Amount of FICSIT Coupons used to purchase (required)   
2 | String | Name of the part or building to be used, defaults to PAGENAME   
3 | String | Replacement image for embedded ItemLink, defaults to {{{1}}}.png   
  
## Example
    
    
    {{Shopunlock|10|Cup}}

**[](/wiki/Cup "Cup")[Cup](/wiki/Cup "Cup") is unlocked via the [](/wiki/AWESOME_Shop "AWESOME Shop") [AWESOME Shop](/wiki/AWESOME_Shop "AWESOME Shop") using:**  
[](/wiki/FICSIT_Coupon "FICSIT Coupon")10  
---  
      
    
    {{Shopunlock|4|Beams|Metal Beam.png}}

**[](/wiki/Beams "Beams")[Beams](/wiki/Beams "Beams") is unlocked via the [](/wiki/AWESOME_Shop "AWESOME Shop") [AWESOME Shop](/wiki/AWESOME_Shop "AWESOME Shop") using:**  
[](/wiki/FICSIT_Coupon "FICSIT Coupon")4
