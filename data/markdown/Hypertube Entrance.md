The **Hypertube Entrance** is a [building](/wiki/Building "Building") used to enter the [Hypertube](/wiki/Hypertube "Hypertube"). Unlike the [Hypertube Support](/wiki/Hypertube_Support "Hypertube Support"), its height or angle cannot be changed during construction. It can also be attached to a Support if it's at the end of the tube. 

A Hypertube can be only attached to it from one side and so has to be built that the [Indicator Light](/wiki/Indicator_Light "Indicator Light") is on the right side from the direction it will be entered from. An arrow shown in the placement hologram shows the entry direction. 

Each Hypertube Entrance constantly consumes 10 MW, but only has to be powered when entering the tube. If a power trip occurs when the Pioneer is already within the tube, transport will not be interrupted. As Hypertubes don't transmit power, each Hypertube Entrance has to be powered to enter (but not to exit). The entry range is about 2 m to the front and 0.5 m to each side. 

Hypertube Entrances can be chained to construct a [Hypertube Cannon](/wiki/Tutorial:Hypertube_cannon "Tutorial:Hypertube cannon"). 

Hypertube Entrances can be [Nudged](/wiki/Controls#Default_keybindings "Controls") (by using the Arrow Keys)

## Indicator Light behavior

Color | State   
---|---  
Red | Lacking power   
Yellow | Powered, no Hypertube attached   
Green | Operating   
  
## Trivia

  * The direction of travel can be reversed during a journey by holding the 'Backward' key
  * It has no encroaching hitbox, so it is possible to build multiple Hypertube Entrances at the same spot.
