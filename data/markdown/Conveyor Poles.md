**Conveyor Poles** are logistic structures used to support [Conveyor Belts](/wiki/Conveyor_Belts "Conveyor Belts") and [Conveyor Lifts](/wiki/Conveyor_Lift "Conveyor Lift"). 

All kinds of Poles allow a Conveyor to be attached to it from both sides, as long as the Conveyor is set to transport in the same direction, which can be especially useful to force the direction of Conveyor Lifts. Conveyors keep on working even if the Poles supporting them are dismantled. 

Conveyor Lifts can be built standalone without supports, unlike Belts.

## Construction

Standard Conveyor Poles are built by clicking ([](/wiki/File:Keyboard_White_Mouse_Left.png "Left")) once to set the location, then scrolling to rotate and looking up or down to change the height (if desired), confirming the placement by clicking again. 

Stackable Conveyor Poles are built simply by placing them on the ground, on each other or other kinds of stackable supports. It's not possible to make them the default type of Poles used when building Conveyor Belts. Simple Conveyor Poles fit under Stackable Poles. Stackable Conveyor Poles can be built one at a time or 10 at once using the [Zoop](/wiki/Build_Gun#Zoop "Build Gun") construction mode. 

Conveyor Wall Mounts can only be built on the sides of Walls and Foundations, as mentioned above. It's not possible to place them on ceilings, unlike [Pipeline](/wiki/Pipeline_Supports "Pipeline Supports") or [Hyper Tube Wall Supports](/wiki/Hyper_Tube_Supports "Hyper Tube Supports"). For that purpose, Conveyor Ceiling Mounts can be used. 

Conveyor Ceiling Mounts now allow for granular rotation when holding the Snap To Guidelines [`Left Ctrl`](/wiki/Controls "Controls") Key. 

**⭑ NOTE:** If the [Build In One Click](/wiki/Settings#General_4 "Settings") option is enabled, [](/wiki/File:Keyboard_White_Mouse_Left.png "Left") **has to be held down** to adjust the rotation and height while placing.
