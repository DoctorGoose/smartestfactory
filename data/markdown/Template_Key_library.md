It serves as a reference for the "proper" names for each console and key for when the user is having trouble figuring out what a button or console is called, but there are additional "improper" terms not shown here that will still work, such as "Xbone" instead of Xbox One. 

Buttons here are displayed at 50px for viewing purposes, but the default size is 20px.
