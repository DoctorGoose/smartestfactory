This template is used to display achievements. Achievements are taken from [Template:Achievements.json](/wiki/Template:Achievements.json "Template:Achievements.json"). 

## Usage

This template should be used in the Achievements section of pages. 

### Parameters

Parameter | Data type | Description   
---|---|---  
1 | String | Internal name of the achievement (required)   
  
## Example

`{{Achievement|COMPLETE_ONBOARDING}}`
