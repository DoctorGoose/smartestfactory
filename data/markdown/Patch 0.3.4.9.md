# Patch 0.3.4.9

Patch Notes: Early Access (EXPERIMENTAL) – **v0.3.4.9** – Build 121997. This patch was released on 13 May 2020.   
  
The original post can be viewed on Satisfactory's [Discord](https://discord.com/channels/370472939054956546/557603412565426196/710154919344209920) and [Reddit](https://www.reddit.com/r/SatisfactoryGame/comments/gj1er2). 

## Contents

  * 1 Intro
  * 2 Resources
  * 3 UI
  * 4 World
  * 5 Bug fixes



## Intro

Hi Pioneers! 

Another update is coming to Experimental. In this one we’ve packed a bunch of client fixes, some more UI improvements, and other fixes for crashes and bugs you can find in the patch notes below. 

We also largely reverted to the old resource meshes. While the new ones improved the readability, they didn’t incorporate what was working well with the old resource meshes. When we change something in the game, the outcome should generally be better than what we had before and that wasn’t the case. The old resource icons did get a little bit of polish, so that they are still more distinguishable in this version. We will look at the resource meshes again in the future and see if they still need some more work. 

Thank you all for the feedback and I hope you have a wonderful day <3 

## Resources

  * Switched back to old resource meshes (except for [Limestone](/wiki/Limestone "Limestone")) and improved icon readability of the old meshes



## UI

  * Added some basic feedback for the Building Sampler / Eyedropper tool
  * Added a keybinding hint for Mass Dismantle in the [Dismantle Mode](/wiki/Dismantle "Dismantle")
  * Updated the look for the Join Session by ID UI in the main menu



## World

  * Added some pick-up SFX and VFX for foliage that was missing it



## Bug fixes

  * [Power Liness](/wiki/Power_Lines "Power Lines") should be easier to dismantle again
  * Fixed a crash related to the replication graph
  * Client players can now properly interact with [Lizard Doggo](/wiki/Lizard_Doggo "Lizard Doggo") inventories
  * Client players can now see [Gas Mask](/wiki/Gas_Mask "Gas Mask") filter consumption
  * Client players can now see the [Space Elevator](/wiki/Space_Elevator "Space Elevator") animations
  * Client players can now interact with [Trains](/wiki/Electric_Locomotive "Electric Locomotive") when the host is far away
  * [The Hub](/wiki/The_HUB "The HUB") is no longer invisible for a short moment when completing the final upgrade
  * [Train Stations](/wiki/Train_Station "Train Station") can now be dismantled and rebuilt in the same spot when adjacent to [Foundations](/wiki/Foundations "Foundations")
  * Train Station clearance is fixed so Foundations can be built around it properly
  * Trains and [Freight Cars](/wiki/Freight_Car "Freight Car") now snap better to each other at track intersections and turnouts
  * Fixed some missing or wrongly scaled Limestone node meshes
  * Fixed crash when trying to search for an online session ID when you are not logged in
