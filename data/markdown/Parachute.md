## Usage

The Parachute is used by pressing the **jump** button (default [`Space`](/wiki/Controls "Controls")) while **descending** in midair. Pressing jump while ascending will not work. 

The Parachute does not get consumed when used, so the player can make one and use it for as long as they want. 

The parachute has a standard and sprint speed, which match the [Blade Runners](/wiki/Blade_Runners "Blade Runners") speeds. 

### Parachute railgun

Shortly after [Update 8](/wiki/Patch_0.8.0.0 "Patch 0.8.0.0") was released for Experimental (EX) / Beta Branch, the use of Parachutes led to the discovery and development of **Parachute Railguns (PRGs)** by using [Beams](/wiki/Beams "Beams") (Painted or Metal) constructed at an angle vertically.[1][2]

While wearing / deploying a Parachute **any vertical or incline surface** will shoot the Pioneer up very fast. This behavior was **not intended** but will be allowed to stay in the game and not fixed.[3]

The use of PRGs is a good way prior to Tier 7 to get a gods-eye view of your local area with relative safety and without using tall cliffs or having to build a [Lookout Tower](/wiki/Lookout_Tower "Lookout Tower") _(or any other tall structure)_ , or use a [Ladder](/wiki/Ladder "Ladder"), or [Jetpack](/wiki/Jetpack "Jetpack"). In Tier 7 most will gravitate to using [Hover Packs](/wiki/Hover_Pack "Hover Pack") to get overhead views of the local area. 

### AWESOME Sink

**[](/wiki/Parachute "Parachute")Parachute** can be sunk into the [](/wiki/AWESOME_Sink "AWESOME Sink") [AWESOME Sink](/wiki/AWESOME_Sink "AWESOME Sink") to yield **6,080 points**.

## History

  * [Patch 1.0.0.4](/wiki/Patch_1.0.0.4 "Patch 1.0.0.4"): Added equip/un-equip sounds for [Blade Runners](/wiki/Blade_Runners "Blade Runners"), [Gas Mask](/wiki/Gas_Mask "Gas Mask") and Parachute
  * [Patch 1.0.0.3](/wiki/Patch_1.0.0.3 "Patch 1.0.0.3"): Fixed some visual issues with the [Build Gun](/wiki/Build_Gun "Build Gun") when deploying the Parachute
  * [Patch 0.8.0.3](/wiki/Patch_0.8.0.3 "Patch 0.8.0.3"): 
    * Fixed the Parachute visuals on Conveyor Belt or when dropped
    * Toned down the new parachute recipe
    * Fix/Removed the option to purchase Parachutes from the AWESOME Shop now that it is no longer a consumable item
  * [Patch 0.8.0.0](/wiki/Patch_0.8.0.0 "Patch 0.8.0.0"): 
    * The Parachute is no longer consumed after usage and the stack size is now 1, making it function the same as other back slot equipment, such as the Jetpack
    * Added a Sprint Mode to the Parachute, allowing for slow and fast movement which match the Blade Runner speeds
    * Increased the maximum speed and descending speed, allowing for improved gliding
    * Changed the Parachute recipe to only produce 1 at a time and increased the cost of a single Parachute
    * Removed the option to purchase Parachutes from the AWESOME Shop.
  * [Patch 0.5.0.4](/wiki/Patch_0.5.0.4 "Patch 0.5.0.4"): Fixed a crash when using Parachutes as [Multiplayer](/wiki/Multiplayer "Multiplayer") / [Dedicated Servers](/wiki/Dedicated_Servers "Dedicated Servers") Client
  * [Patch 0.3.4.8](/wiki/Patch_0.3.4.8 "Patch 0.3.4.8"): Parachute should be usable for clients now
  * [Patch Closed Alpha 5.1](/wiki/Patch_Closed_Alpha_5.1 "Patch Closed Alpha 5.1"): Fixed entire Parachute stack being consumed, now only consumes one
