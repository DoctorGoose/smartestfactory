The Ping Function has been in the game since pre-Closed Alpha[1] with the notable change from using an "X" character for the "Ping" to using an exclamation mark "!" character[2] for the "Ping" at the Ping location. 

The player can place a **Ping** at the location where the cursor is by pressing [`Alt`](/wiki/Controls "Controls") \+ Left Mouse Button [](/wiki/File:Keyboard_White_Mouse_Left.png "Left"). The ping indicator shows up as an exclamation mark and is now visible on the [on-screen Compass](/wiki/HUD#Compass "HUD") and [Map](/wiki/Map "Map"). The ping also makes a sound coming from the ping location. Pings can be used to point at things for other players to see.

## Current issues

  * Sometimes the player can get stuck in ping mode after using [`Alt`](/wiki/Controls "Controls") \+ [`Tab ↹`](/wiki/Controls "Controls") to switch windows out of the game, resulting in every click being interpreted as a ping regardless of if [`Alt`](/wiki/Controls "Controls") is pressed. Press [`Alt`](/wiki/Controls "Controls") again to resolve this, returning clicking functionality to normal.



## Trivia

Ping can be used to survey terrain distance, although it has a limited range ~500 metres.
