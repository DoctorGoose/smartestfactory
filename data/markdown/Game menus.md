# Game menus

_Satisfactory_ utilizes several types of **menus** , most notably the main menu (title screen) and escape menu. 

Some other examples include the [Inventory](/wiki/Inventory "Inventory") ([`Tab ↹`](/wiki/Controls "Controls")), radial menus such as the [Resource Scanner](/wiki/Resource_Scanner "Resource Scanner"), building configuration menus, etc.

## Main menu

The main menu is an introductory interface seen when the game starts. Prominently featuring the _Satisfactory_ logo and a thematic background, it allows a game session to start by creating a new one or loading a [save file](/wiki/Save_file "Save file"). It also gives access to the [settings](/wiki/Settings "Settings") menu. 

### Menu options

Button  | Usage  | Notes   
---|---|---  
Continue  | Loads the most recent save file, starting a game session.  | Only available if any save file exists   
New Game  | Brings up the [New Game Start Screen](/wiki/New_game "New game"), allowing a new session to be started.  | Starts new game session. Both [Multiplayer Settings](/wiki/Multiplayer "Multiplayer") and [Advanced Game Settings](/wiki/Advanced_Game_Settings "Advanced Game Settings") can be enabled before starting.   
Join Game  | Allows the Multiplayer Clients to join another Players current Multiplayer Session. See the linked page for details.  | See also [Multiplayer](/wiki/Multiplayer "Multiplayer")  
Load  | Loads the selected save file, starting a game session. [Advanced Game Settings](/wiki/Advanced_Game_Settings "Advanced Game Settings") can be enabled before loading.  
Save files can also be deleted in this menu.  |   
Options  | Allows access to the settings menu.  | See also [Settings](/wiki/Settings "Settings")  
Online Settings  | Allows accepting invites from other players.  | See also [Multiplayer](/wiki/Multiplayer "Multiplayer").   
Server Manager  | Allows the management of Dedicated servers.  | See also [Dedicated servers](/wiki/Dedicated_servers "Dedicated servers")  
Mods  | Allows access to Mod configuration files.  | ⭑ _Menu option only appears if Mods are loaded._  
Credits  | Shows game credits.  |   
Exit  | Closes the game and exits to desktop.  |   
  
### Current game info

Below the menu options the following current game information overlay is displayed: 

  * **Player Account and Name** \- Shows the Player Account (Steam or Epic) and Player Account Name.
  * **Version** \- Shows current [build number](/wiki/Category:Patch_notes "Category:Patch notes").



#### Game mods loaded

In addition to the current game information, the following will also be shown if any game mods are being used: 

  * **Mod Loader** \- Shows current version of the [Satisfactory Mod Manager (SMM)](https://docs.ficsit.app/satisfactory-modding/latest/ForUsers/SatisfactoryModManager.html) and [Satisfactory Mod Loader (SML)](https://ficsit.app/mod/SML) used. 
    * The SML is a game mod that is automatically added and updated via the SMM.
  * **Mods** \- Shows the number of game mods that are currently loaded.



### Patch notes

To the right on the Main Menu the latest [Patch Notes](/wiki/Category:Patch_notes "Category:Patch notes") will be shown in abbreviated form. Click **Read More** will show the full version of latest patch notes.

### Offline warning

Starting an offline session will show a popup to inform them that an online session could not be created. You can retry the sign-in process by restarting the game.

## Escape menu

The in-game escape menu, as the name suggests, is shown by pressing [`Esc`](/wiki/Controls "Controls"). 

### Menu options

Button  | Usage  | Notes   
---|---|---  
Resume Game  | Closes the menu.  |   
Save Game  | Allows saving the current session by creating a save file.  |   
Load Game  | Allows loading a different save file, starting a previous game from current session, or starting a different game session.  | [Advanced Game Settings](/wiki/Advanced_Game_Settings "Advanced Game Settings") can be enabled before loading one.   
Manage Session  | Shows a list of currently connected players, allows changing the session privacy, etc.  | See also [Multiplayer](/wiki/Multiplayer "Multiplayer")  
Go to feedback site  | Opens the [Q&A site](https://questions.satisfactorygame.com) in the default web browser.  |   
Options  | Allows access to the settings menu.  | See also [Settings](/wiki/Settings "Settings")  
Server Manager  | Allows the management of Dedicated servers.  | See also [Dedicated servers](/wiki/Dedicated_servers "Dedicated servers")  
Mods  | Allows access to Mod configuration files.  | ⭑ _Menu option only appears if Mods are loaded._  
Mod Savegame Settings  | Current Mod configuration files settings.  | ⭑ _Menu option only appears if Mods are loaded._  
Exit to Main Menu  | Exits to the main menu.  | The game is **not** saved automatically   
Exit to Desktop  | Closes game and exits to desktop.  | The game is **not** saved automatically   
Respawn | Instantly kills your character. This allows you to respawn if stuck.  | **There is no confirmation prompt.** Other players in [multiplayer](/wiki/Multiplayer "Multiplayer") can still revive you.   
  
### Current game info

Below the menu options the following current game information overlay is displayed: 

  * **Current Session** \- Shows name of the Game Save that is currently loaded.
  * **Player Account and Name** \- Shows the Player Account (Steam or Epic) and Player Account Name.
  * **Version** \- Shows current [build number](/wiki/Category:Patch_notes "Category:Patch notes").



#### Game mods loaded

In addition to the current game information, the following will also be shown if any game mods are being used:

    * The SML is a game mod that is automatically added and updated via the SMM.
  * **Mods** \- Shows the number of game mods that are currently loaded.

## History

  * [Patch 1.1.1.2](/wiki/Patch_1.1.1.2 "Patch 1.1.1.2"): 
    * Potential fix for NullPTR crash when closing the game
    * Fixed a crash related to FGBackgroundThread
  * [Patch 1.1.1.1](/wiki/Patch_1.1.1.1 "Patch 1.1.1.1"): Fixed Welcome Screen showing up on every launch on Steam
  * [Patch 1.0.0.4](/wiki/Patch_1.0.0.4 "Patch 1.0.0.4"): Fixed long text not properly scaling in some of the menus
  * [Patch 1.0.0.3](/wiki/Patch_1.0.0.3 "Patch 1.0.0.3")
    * Fixed multiple crashes that could happen when closing the game
    * Fixed a bug where collapsable lists in the [Options Menu](/wiki/Settings "Settings") would sometimes be expanded when exiting and re-entering menus or using the search function
  * [Patch 1.0](/wiki/Patch_1.0 "Patch 1.0")
    * Changed Main Menu background image and animation
    * Changed In-Game Pause / Escape Menu
    * Undocumented change. Removed the red ⚠ warning symbol for Server Manager menu item
  * [Patch 0.8.2.9](/wiki/Patch_0.8.2.9 "Patch 0.8.2.9")
    * Added a popup to inform when an online session could not be created and that you are about to continue offline 
      * You can retry the sign-in process by restarting the game
  * [Patch 0.8.2.4](/wiki/Patch_0.8.2.4 "Patch 0.8.2.4"): Added additional logging to help debug an issue where some players are unable to write a save to their computer due to the process failing
  * [Patch 0.8.2.1](/wiki/Patch_0.8.2.1 "Patch 0.8.2.1")
    * Main Menu buttons should no longer be greyed out when the online integration is not fully initialized (When online or offline)
    * Removed the loading screen related to the online functionality, This will help for players that get stuck on the loading screen, it also fixes the Account linking popup never showing up
  * [Patch 0.8.2.0](/wiki/Patch_0.8.2.0 "Patch 0.8.2.0"): News Feed in the Main Menu should now be easier to read
  * [Patch 0.8.0.0](/wiki/Patch_0.8.0.0 "Patch 0.8.0.0")
    * Changed Advanced Game Settings menu option 
      * Menu option opens new [Advanced Game Settings](/wiki/Advanced_Game_Settings "Advanced Game Settings") menu
      * Undocumented change - Creature Hostility moved to [Options > Gameplay](/wiki/Settings#Gameplay_2 "Settings") [1]
  * [Patch 0.7.0.0](/wiki/Patch_0.7.0.0 "Patch 0.7.0.0")
    * Implemented double click functionality for Save and Load Game
    * Added a new submenu called Multiplayer Settings to the Escape Menu and moved Manage Session/Manage Invites there
    * Added a cancel button to "Add new server" popup in the Server Manager
    * Added a search bar to the multiplayer list in the Join Game menu
    * Added Creature hostility modes that can be set in Advanced Game Settings in the Esc Escape Menu 
      * Regular: Creatures attack and investigate players based on vision and hearing
      * Retaliate: Creatures attack players after being attacked themselves
      * Passive: Creatures never attack players
  * [Patch 0.6.0.14](/wiki/Patch_0.6.0.14 "Patch 0.6.0.14"): Potential fix for softlock when respawning with a menu open
  * [Patch 0.6.0.10](/wiki/Patch_0.6.0.10 "Patch 0.6.0.10")
    * Fixed key shortcuts showing the wrong shortcuts in some menus
    * Polished the New Game Starting Area selection button
  * [Patch 0.6.0.5](/wiki/Patch_0.6.0.5 "Patch 0.6.0.5")
    * Fixed not being able to save without typing a name for the session first (Name will be auto set to “SessionName_Timestamp” if the name field is blank)
    * Fixed Delete Save button not working properly
  * [Patch 0.6.0.0](/wiki/Patch_0.6.0.0 "Patch 0.6.0.0"): Optimized the Save and Load UI
  * [Patch 0.5.2.0](/wiki/Patch_0.5.2.0 "Patch 0.5.2.0"): Fixed a bug when starting a game with the “Skip Intro” option and then loading a save would result in the intro also being skipped in that save unless exiting the game and relaunching it first
  * [Patch 0.5.1.0](/wiki/Patch_0.5.1.0 "Patch 0.5.1.0"): News feed (Main Menu) background is now darker and semi transparent for better readability
  * [Patch 0.5.0.12](/wiki/Patch_0.5.0.12 "Patch 0.5.0.12"): Fixed “START GAME” Button being off screen in the New game menu
  * [Patch 0.5.0.11](/wiki/Patch_0.5.0.11 "Patch 0.5.0.11"): News feed (Main Menu) background is now darker and semi transparent for better readability
  * [Patch 0.5.0.0](/wiki/Patch_0.5.0.0 "Patch 0.5.0.0")
    * Added Server Manager option for [Dedicated servers](/wiki/Dedicated_servers "Dedicated servers") to Main Menu and In-game Pause / Escape menu
    * Undocumented change - Network Address Translation (NAT) used info moved from Main Menu to In-game Pause / Escape menu
  * [Patch 0.3.5.1](/wiki/Patch_0.3.5.1 "Patch 0.3.5.1"): Undocumented change - The following was noted:
