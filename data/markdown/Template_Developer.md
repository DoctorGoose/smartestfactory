## Usage

Edit this template to replace the name of the game developer in the [legal notices](/wiki/Category:Legal_notices "Category:Legal notices") and [copyright template](/wiki/Template:Copyright_game "Template:Copyright game"). If you wish to also link to the developer's site, you can specify the site url as follows: 

    `{{Developer|<http://game.site.com>}}`

Result: 

    [Coffee Stain Studios](http://game.site.com)

Alternatively, you can simply remove the logic and hardcode the url in the template, rather than using the unnamed variable: `{{{1}}}`. This will link to the developer's site in all instances, rather than optionally.
