# Priority Power Switch

## Priority Power Switch  
  
[ ](/wiki/File:Priority_Power_Switch.png "Priority Power Switch.png")

_Priority Power Switches can be ranked by priority. When power production is too low, Switches will start turning off automatically until the power stabilizes, starting with Priority Group 8._

The **Priority Power Switch** is an improved variant of the [Power Switch](/wiki/Power_Switch "Power Switch"). It can be used to set which sections (named Circuit Break Priority) [1] of the [power grid](/wiki/Power_grid "Power grid") should shut off first, if a power trip is about to occur. Additionally, it allows other Priority Power Switches on the grid to be toggled remotely.

## Usage

Any Priority Power Switch in the world can be remotely turned on or off from the user interface of any other Priority Power Switch. The user interface has two tabs. The Priority Power Switch can be named in the _Priority Power Switch_ tab. 

In the _Circuit Break Priority_ tab, **new switches appear in the "Undefined" priority group.** Named switches can be dragged from one priority group to another. 

When there is a power outage on a [power grid](/wiki/Power_grid "Power grid"), Priority Power Switches will turn off, starting with Priority Group 8, until power supply is adequate to meet demand. Priority Group 1 will turn off last. Pioneers are advised to put power generating plants and their fuel production chains in group one. 

**⭑ NOTE:** Any Priority Power Switches in the "Undefined" priority group will turn off first, as "Undefined" is considered the lowest priority, followed by Priority Group 8.
