# Template:MAMusage

[](/wiki/File:Template-info.svg) Documentation

[[purge](https://satisfactory.wiki.gg/wiki/Template:MAMusage?action=purge)]

The MAMusage template is used in the Usage section of parts under Research. It displays that the part is used in the M.A.M. in a certain research chain (or chains), or that it unlocks a research chain. 

## Usage

### Parameters

Parameter | Data Type | Description   
---|---|---  
newchain | Any | Entering any non-blank value will make the template show that the part unlocks a certain research chain rather than that it is used in one. If used, can only display one chain (chain1)   
chain1 | String | Name of the research chain, without "research" or "research chain" (required)   
chain2 | String | Name of the second research chain, without "research" or "research chain"   
chain3 | String | Name of the third research chain, without "research" or "research chain"   
chain4 | String | Name of the fourth research chain, without "research" or "research chain"   
chain5 | String | Name of the fifth research chain, without "research" or "research chain"   
name | String | Name of the part to be used, if left blank pagename is used   
  
## Example
    
    
    {{MAMusage
    |chain1 = Sulfur
    }}
