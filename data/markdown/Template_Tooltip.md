# Template:Tooltip

[](/wiki/File:Template-info.svg) Documentation

[[purge](https://satisfactory.wiki.gg/wiki/Template:Tooltip?action=purge)]

Template to easily add a tooltip 

## Usage
    
    
    {{Tooltip|Text shown normally|Text shown on hover|noUnderline}}

## Example

`{{Tooltip|Dodos are tasty|May require seasoning}}`  
`{{Tooltip|Dodos are tasty|May require seasoning|Any non empty text}}`

### Result

Dodos are tasty  
Dodos are tasty
