# Supercomputer

## Supercomputer

[ ](/wiki/File:Supercomputer.png "Supercomputer.png")

_The next-gen version of a Computer._

### Unlocked by

[Tier 7](/wiki/Tier_7 "Tier 7") \- Control System Development

### Class name

Desc_ComputerSuper_C

## Item

### Stack size

50

### [Sink points](/wiki/AWESOME_Sink "AWESOME Sink")

97,352

The **Supercomputer** is a late-game component used to construct [Nuclear Power Plants](/wiki/Nuclear_Power_Plant "Nuclear Power Plant"), [Particle Accelerators](/wiki/Particle_Accelerator "Particle Accelerator") and other late game buildings, as well as [Assembly Director Systems](/wiki/Assembly_Director_System "Assembly Director System") and [Neural-Quantum Processors](/wiki/Neural-Quantum_Processor "Neural-Quantum Processor").

### AWESOME Sink

**[](/wiki/Supercomputer "Supercomputer")Supercomputer** can be sunk into the [](/wiki/AWESOME_Sink "AWESOME Sink") [AWESOME Sink](/wiki/AWESOME_Sink "AWESOME Sink") to yield **97,352 points**.

## Trivia

  * There is an old Supercomputer mesh below the terminals in [the HUB](/wiki/The_HUB "The HUB")[1] and the [Space Elevator](/wiki/Space_Elevator "Space Elevator").[2]
  * The mesh size of the computer present in the old Supercomputer mesh is slightly larger than the model of the [Computer](/wiki/Computer "Computer") alone.



## Current issues

Supercomputer is available in the AWESOME Shop after completion of [Tier 8, Milestone 1 - Aeronautic Engineering](/wiki/Milestones#Tier_8 "Milestones") rather than [Tier 7, Milestone 5 - Control System Development](/wiki/Milestones#Tier_7 "Milestones"). See: [Q&A Post](https://questions.satisfactorygame.com/post/66fe7d0e772a987f4a8e5843).
