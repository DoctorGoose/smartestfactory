**Hypertube Supports** are logistic structures used to support [Hypertubes](/wiki/Hypertube "Hypertube"). 

All kinds of Supports allow a Hypertube to be attached to it from both sides, as well as [Hypertube Entrances](/wiki/Hypertube_Entrance "Hypertube Entrance"). Hypertubes keep on working even if the Supports supporting them are dismantled.

## Construction

Standard Hypertube Supports are built by clicking ([](/wiki/File:Keyboard_White_Mouse_Left.png "Left")) once to set the location, then scrolling to rotate (or [`Ctrl`](/wiki/Controls "Controls") angle) and looking up or down to change the height (if desired), confirming the placement by clicking again. If the "[Execute Build Step on Release](/wiki/Settings "Settings")" option is enabled, [](/wiki/File:Keyboard_White_Mouse_Left.png "Left") has to be held down to adjust the rotation and height while placing. 

Stackable Hypertube Supports are built simply by placing them on the ground, on each other or other kinds of stackable supports. It's not possible to make them the default type of Supports used when building Hypertubes. Stackable Hypertube Supports can be built one at a time or 10 at once using the [Zoop](/wiki/Build_Gun#Zoop "Build Gun") construction mode. 

Hypertube Wall Supports can only be built on Walls and on the sides and ceilings of Foundations, as mentioned above. Hypertube Wall Holes can be built on Walls and serve the same purpose as Conveyor Walls, but can be placed anywhere on the Wall, making them much more modular.
