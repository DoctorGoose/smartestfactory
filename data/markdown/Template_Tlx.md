# Template:Tlx

[](/wiki/File:Template-info.svg) Documentation

[[purge](https://satisfactory.wiki.gg/wiki/Template:Tlx?action=purge)]

This template behaves like `{{[Tl](/wiki/Template:Tl "Template:Tl")}}` but allows the parameters to be displayed as well, for example: 

  * `{{tlx|doc/start|clear}}` generates `{{[doc/start](/wiki/Template:Doc/start "Template:Doc/start")|clear}}`.
