# Signs

## Signs

[ ](/wiki/File:Display_Sign.png "Display Sign.png")

_Signs can be customized with different elements, such as icons and text, to improve factory organization.  
Can be placed freely, on Walls, or attached to a variety of buildings, such as Storage Containers._

### Unlocked by

[AWESOME Shop](/wiki/AWESOME_Shop "AWESOME Shop")

**Signs** are organizational structures that can be placed freely or attached to most structural and factory buildings, such as [Walls](/wiki/Walls "Walls"), [Foundations](/wiki/Foundations "Foundations"), [Storage Containers](/wiki/Storage_Container "Storage Container"), [Truck Stations](/wiki/Truck_Station "Truck Station") etc. 

They can be customized with different elements, such as icons and text.

## Construction

Signs can attach to the ground, walls, ceilings, and predetermined snapping points on buildings. The exact options depend on the size of the sign; Billboards especially cannot attach to as many snapping points as other signs. Some signs may be clipped into others to create more complex signs. 

Signs may be placed in different orientations such as vertically, on floors, or on ceilings by placing them on rotated [beams](/wiki/Beams "Beams") and the end of [pillars](/wiki/Pillars "Pillars"). When snapping a sign to a [foundation](/wiki/Foundation "Foundation"), ceiling or floor, you can hold [`Ctrl`](/wiki/Controls "Controls") to lay the sign down flat, otherwise it will use the pole support. 

The **sign pole** can be copied using the [Building Eyedropper Function](/wiki/Build_Gun#Building_eyedropper "Build Gun"), and built separately wherever you wish. You can even adjust the length of the copied sign pole during placement elsewhere.[1] Additionally the sign pole can be deleted after placement of the Sign.

### General

The Sign Designer can be accessed by interacting [`E`](/wiki/Controls "Controls") with any sign. The following customization options are available. Label and Square signs have fewer options. 

  1. **Colors:** Clicking _Select Colors_ opens a [Color Manager](/wiki/Customizer#Color_Manager "Customizer") _(similar to that used in the Customizer)_ , where the top sliders are for the text color, and the bottom sliders are for the background color.
  2. **Icon:** Most [items](/wiki/Items "Items"), [fluids](/wiki/Fluids "Fluids"), [buildings](/wiki/Buildings "Buildings"), [vehicles](/wiki/Vehicles "Vehicles"), as well as some monochrome icons, [FICSIT](/wiki/FICSIT "FICSIT") and [FICSMAS](/wiki/FICSMAS "FICSMAS") branding can be used as the sign icon.
  3. **Background:** Behind text and images, some are animated.
  4. **Text:** Change text using this field. Default text is "Shennanigans" (sic).
  5. **Layout:** several layouts are available, such as icon+text, 2 icons+text, text only, large text, icon only etc.
  6. and 7: **Copy Settings / Paste Settings:** Allow player to copy and paste sign customization between signs.


  1. **Revert Settings:** Allows player to reset sign settings to default.
  2. **Material settings:** _(not in shown on right)_



    

  * _Emission Strength_ : Adds a glow to the sign (intensity 0–3).
  * _Surface Finish_ : Allows sets the finish to flat (Matte) or reflective (Glossy),



### Signs as lights

Signs can be used to add lighting and accents, in addition to, or in place of [Lights](/wiki/Lights "Lights"). To do this, in the Sign Designer, do the following:[2][3]

  * Remove the default **Icon** and **Text**.
  * Set the **Color** for the _background_ to the color desired. There is no need to change the _text color_ unless it is desired to do so.
  * Set the **Emission Strength** to desired level _(but normally 2 or 3 is commonly used)_.



### Global illumination (lumen)

Often signs, especially those used for lighting and accent, won't effectively illuminate close by areas when [Global Illumination (Lumen)](/wiki/Settings#Advanced "Settings") is set to _Medium_ or _High_ despite the setting for **Lighting Mood**. To resolve this, the use of the following [Debug console](/wiki/Debug_console "Debug console") commands may help: 

  * `r.AOGlobalDistanceField.MinMeshSDFRadius [number]` where [number] _(without brackets)_ is a whole number value of 1 or higher. Values up to 20 seem to work in various degrees. This command provides a temporary fix to make emissive materials _(like Signs)_ in Global Illumination (Lumen) illuminate further.
  * `r.LumenScene.SurfaceCache.CardTexelDensityScale [number]` where [number] _(without brackets)_ is a whole number value of 1 or higher. Values up to 3000 seem to work fine. May decrease performance. This command provides a temporary fix to increase render distance of emissive Signs.



Debug Console commands can be made to activate upon game load by modifying the game settings file directly. 

  * See [Debug Console > Options not available in the settings menu](/wiki/Debug_console#Options_not_available_in_the_settings_menu "Debug console") for more information.
