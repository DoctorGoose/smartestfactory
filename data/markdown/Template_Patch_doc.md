[](/wiki/File:Template-info.svg) Documentation

[[purge](https://satisfactory.wiki.gg/wiki/Template:Patch/doc?action=purge)]

This template is used for the mean section on patch note pages. 

## Parameters

Parameter | Type | Details   
---|---|---  
stable | Boolean | Used for non-early-access "stable" [1.0, ) builds   
ex | Boolean | Determines whether the patch was released on experimental   
version | String | Defaults to PAGENAME without 'Patch', sets the version number   
build | String | Build number   
date | Date | Date of the patch's release, format YYYY-MM-DD, UTC   
text | String | Text to place after the first line, but before the second line with links   
website | String | Website post URL   
discord | String | Discord message URL ([](/wiki/File:Keyboard_White_Mouse_Right.png "Right") Copy Message Link, the first one if there are multiple)   
reddit | String | Reddit post ID (usually 6 characters, comes from the URL as reddit.com/r/SatisfactoryGame/comments/**ID**)   
steam | String | Steam news post URL (via the Share button at the bottom)   
nocategory | Boolean | Whether to categorize the page
