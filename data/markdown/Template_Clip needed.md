# Template:Clip needed

  * Tells a visitor that the statement is not yet supported by a reference to a clip from a livestream VOD or video
  * Displays [_Clip needed_] when given no description parameter
  * Displays [_Clip needed_] when given a description parameter
  * Editors are encouraged to use the description parameter (e.g. {{clip needed|DESCRIPTION GOES HERE}}) to indicate where a clip may be sourced from.
  * If no clip is known, vaguer date ranges should be used but too much uncertainty should require `{{[citation needed](/wiki/Template:Citation_needed "Template:Citation needed")}}` be used instead.



## Examples

  * {{clip needed}} adds [_[Clip needed](/wiki/Category:Clip_needed "Category:Clip needed")_]
  * {{clip needed|August 31st, 2021 Livestream}} adds [_[Clip needed](/wiki/Category:Clip_needed "Category:Clip needed")_]
