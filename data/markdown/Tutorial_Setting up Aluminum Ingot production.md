## Overview

Aluminum Production is unlocked at [Tier 7](/wiki/Milestones "Milestones") \- Bauxite Refinement. Once unlocked, you can set up Aluminum Ingot production. 

[Aluminum Ingot](/wiki/Aluminum_Ingot "Aluminum Ingot") production is done in 3 steps. The first step is producing [Alumina Solution](/wiki/Alumina_Solution "Alumina Solution"), which requires [Bauxite](/wiki/Bauxite "Bauxite") and [Water](/wiki/Water "Water"), and has a byproduct of [Silica](/wiki/Silica "Silica"). The second step is producing [Aluminum Scrap](/wiki/Aluminum_Scrap "Aluminum Scrap"), which takes [Alumina Solution](/wiki/Alumina_Solution "Alumina Solution") and has a byproduct of [Water](/wiki/Water "Water"). The third step is producing [Aluminum Ingot](/wiki/Aluminum_Ingot "Aluminum Ingot"), which requires [Aluminum Scrap](/wiki/Aluminum_Scrap "Aluminum Scrap") and [Silica](/wiki/Silica "Silica"). 

Reusing waste [Water](/wiki/Water "Water") is the main initial challenge when setting up [Aluminum Ingot](/wiki/Aluminum_Ingot "Aluminum Ingot") production. Piping it back to the original supply requires difficult piping. An alternative is to use waste water as the only source for a single [Alumina Solution](/wiki/Alumina_Solution "Alumina Solution") producer. 

## Minimal setup

Here we focus on building a minimal unit that reuses all of the waste water, and doesn't get stuck. 

### Step 0 - basic resources

Acquire basic resources: 

  * [Bauxite](/wiki/Bauxite "Bauxite") \- 180/m
  * [Water](/wiki/Water "Water") \- 180/m
  * [Coal](/wiki/Coal "Coal") \- 90/m
  * [Silica](/wiki/Silica "Silica") \- 150/m



Over- or under-provisioning any resources will not break the setup. Under-provisioning will slow it down. 

### Step 1 - [Alumina Solution](/wiki/Alumina_Solution "Alumina Solution") production

Build 2 [Refineries](/wiki/Refinery "Refinery") to produce [Alumina Solution](/wiki/Alumina_Solution "Alumina Solution"). Connect water **only to one of them**. The other one will use waste water. Set the other refinery to **50% capacity**. 

### Step 2 - [Aluminum Scrap](/wiki/Aluminum_Scrap "Aluminum Scrap") production

Build 1 [Refinery](/wiki/Refinery "Refinery") to produce [Aluminum Scrap](/wiki/Aluminum_Scrap "Aluminum Scrap"). Set it to **75% capacity**. Connect all the [Alumina Solution](/wiki/Alumina_Solution "Alumina Solution") production and [Coal](/wiki/Coal "Coal") to it. 

### Step 3 - [Aluminum Ingot](/wiki/Aluminum_Ingot "Aluminum Ingot") production

Build 3 [Foundries](/wiki/Foundry "Foundry") to produce [Aluminum Ingot](/wiki/Aluminum_Ingot "Aluminum Ingot"). Connect **one of them to the byproduct[Silica](/wiki/Silica "Silica")** from Step 2. Connect the rest to the external [Silica](/wiki/Silica "Silica") source. 

### Summary

This setup produces total of 180 [Aluminum Ingot](/wiki/Aluminum_Ingot "Aluminum Ingot")/m. 

## Expanding production

The setup described above can be used as a minimal unit (later called a **Production Node**), and repeated side-by-side to increase production. However, some improvements can be made. They save you building space, but not resources nor energy. 

### Improvement 1 - Reuse waste water [Refinery](/wiki/Refinery "Refinery") from Step 1

[Refinery](/wiki/Refinery "Refinery") from Step 1 was set to only 50% capacity. It can be used to recycle water from 2 Production Nodes, saving you one [Refinery](/wiki/Refinery "Refinery"). 

### Improvement 2 - Reuse scrap [Refinery](/wiki/Refinery "Refinery") from Step 2

[Refinery](/wiki/Refinery "Refinery") from Step 2 was set to only 75% capacity. When building 4 Production Nodes, using 3 [Refineries](/wiki/Refinery "Refinery") to produce [Aluminum Scrap](/wiki/Aluminum_Scrap "Aluminum Scrap") will be enough.
