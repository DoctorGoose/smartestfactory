# Patch Closed Alpha 4.1

Patch notes: **Closed Alpha 4.1** – Build 94144 – _Bug fixes & game improvements_. This version was released on March 1, 2019. The original post came from the Closed Alpha Discord.[1]  
  
## Contents

  * 1 Intro
  * 2 Bug fixes
  * 3 World
  * 4 UI
  * 5 Progression
  * 6 Buildings
  * 7 References



## Intro

Here are some more fixes before the weekend! Have a good time :D 

## Bug fixes

  * [Nut](/wiki/Beryl_Nut "Beryl Nut") Bush now uses new meshes and scales correctly
  * Fixed an issue where the options menu wasn't loading correctly
  * Fixed [production indicator](/wiki/Indicator_Light "Indicator Light") on Hub Biomass Burner not showing the right colours
  * Fixed [Storage](/wiki/Storage_Container "Storage Container") rotation bug
  * Fixed that some resource items were not pickupable
  * Fixed a set FOV on [Drop Pod](/wiki/Drop-pod "Drop-pod") mesh
  * Fixed a crash related to destroying foliage
  * Fixed client issues with updating [the Hub](/wiki/The_Hub "The Hub") when upgrading it
  * Removed rogue explorer from the level
  * Fixed Continue button in the Main Menu
  * Fixed that Escape works to navigate back in the Main Menu
  * Fixed production warnings in the Manufacturing and Extractor Menus
  * Fixed focus related crashes
  * Fixed not being able to research in the [M.A.M](/wiki/M.A.M "M.A.M")
  * Potentially fixed a crash related to the [Chainsaw](/wiki/Chainsaw "Chainsaw")
  * Fixed [Oil](/wiki/Crude_Oil "Crude Oil") Node placements



## World

  * More Grass Fields specific combat and exploration tweaks
  * Implemented Grass Fields [Stone](/wiki/Stone "Stone") Node and grass fixes
  * Fixed new grass and rivers in Lake Forest (south of Northern Forest)
  * Fixed new grass and rivers in Maze Canyons (east of Northern Forest)
  * Updated boulder meshes



## UI

  * Added more new icons
  * Hub Menu: Changed “Researching…” to “Current Milestone”



## Progression

  * Renamed Alien Armor Plates to [Alien Carapace](/wiki/Alien_Carapace "Alien Carapace")
  * Renamed enemy drop [research](/wiki/Research "Research") to new names: Alien Organs and Alien Carapace



## Buildings

  * Updated UVs on Hub monitors
