# Patch 0.3.4.11

Patch Notes: Early Access (EXPERIMENTAL) – **v0.3.4.11** – Build 122801. This patch was released on 22 May 2020.   
  
The original post can be viewed on Satisfactory's [Discord](https://discord.com/channels/370472939054956546/557603412565426196/713322131806617600) and [Reddit](https://www.reddit.com/r/SatisfactoryGame/comments/gog56n). 

## Contents

  * 1 Intro
  * 2 Multiplayer
  * 3 Creatures
  * 4 UI
  * 5 Bug fixes



## Intro

Hi Pioneers! 

We’ve got an update for you coming to Experimental that has a bunch of bug fixes in it. You can check them all out in the patch notes below! There are also some smaller UI tweaks and another tweak to tone down the Flying Crabs a little more. 

Our multiplayer connection handling has also changed a bit and is still being worked on. Please let us know if you encounter any issues on the questions site, we will fix them as soon as we can. 

<https://questions.satisfactorygame.com/>

Hope you all are going to have a wonderful weekend <3 

## Multiplayer

  * Implemented changes to multiplayer connection handling and some additional UI. Still work in progress.



## Creatures

  * Increased hitstun on the [Flying Crabs](/wiki/Flying_Crab "Flying Crab") a little more



## UI

  * Changed starting area descriptions and order in the New Game menu
  * Added some reminders that multiplayer has more bugs than single player in relevant menus
  * Show/hide [map](/wiki/Map "Map") filters now saves when closing the menu
  * Moved [Walkways](/wiki/Walkways "Walkways") into the Organization tab in the [AWESOME Shop](/wiki/AWESOME_Shop "AWESOME Shop") to be consistent with the Build Menu



## Bug fixes

  * Small visual tweaks on [Limestone](/wiki/Limestone "Limestone") [node](/wiki/Resource_Node "Resource Node") meshes in the world
  * [Resource Scanner](/wiki/Resource_Scanner "Resource Scanner") icons now correspond with the resource meshes again
  * [Blade Runner](/wiki/Blade_Runners "Blade Runners") [MAM](/wiki/MAM "MAM") research now has an icon
  * [Codex](/wiki/Codex "Codex") onboarding step no longer blocks pick-up input
  * Clients should see less build effects replaying when joining a game
  * Clients now get the correct tutorial info when joining a newly started game
  * Resource extractors now check whether the full resource stack size can be extracted into the output, which will pause production correctly if that is not the case (most noticeable on Oil/Water Extractors)
  * Items will no longer be lost by placing them into a dismantling building’s inventory
  * Items can no longer be duplicated by taking them from a dismantling building’s inventory
  * [Nuts](/wiki/Beryl_Nut "Beryl Nut") and [Berries](/wiki/Paleberry "Paleberry") should now regrow properly
  * Fixed some creature respawn issues
  * [Supercomputer](/wiki/Supercomputer "Supercomputer") research naming is now consistent with the part naming
  * Codex onboarding message now has voice over
  * Fixed some gas areas and encounters in Rocky Desert that were problematic
  * Clients can now empty single [Pipelines](/wiki/Pipelines "Pipelines") and [Fluid Buffers](/wiki/Fluid_Buffer "Fluid Buffer")
  * MAM researches should become available again when picking up the relevant items
  * Clients can now toggle standby correctly in production buildings and resource extractors (still needs to be fixed for generators)
