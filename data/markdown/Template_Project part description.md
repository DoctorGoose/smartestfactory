# Template:Project part description

**Project part description** is a component of [Project Assembly](/wiki/Project_Assembly "Project Assembly"). It is used to complete deliveries in the [Space Elevator](/wiki/Space_Elevator "Space Elevator"), which in return unlocks additional [tiers](/wiki/Milestones "Milestones") in [the HUB](/wiki/The_HUB "The HUB"). As with all [Project Parts](/wiki/Category:Project_Assembly_parts "Category:Project Assembly parts"), it cannot be crafted in a [Crafting Bench](/wiki/Crafting_Bench "Crafting Bench") and has to be automated.
