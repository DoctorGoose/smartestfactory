PackagedFluidInfo is the packaged form of PackagedFluidInfo that can be carried in inventories and transported via Conveyor Belts instead of Pipelines . It is made in Packagers using Empty Canisters , where it can be also unpackaged, refunding the used Canisters.

Documentation
[ purge ]

Used for a unified description of all packaged fluids. It has one optional parameter, the name of the packaged item, which defaults to the page name. The template automatically removes "Packaged " from the name in the description.
