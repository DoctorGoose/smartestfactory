This template makes use of ParserFunctions in the MediaWiki software to calculate the amount of days past since a specified date. Its main usage is on the Main Page to display how long the latest patch, test patch, blog post and version has been released.

Usage

{{ Days ago }}

Example

  • {{ Days ago }} generates 5442 days ago
