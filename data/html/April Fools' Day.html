Starting on April 1st, 2021, and also on the same date in subsequent years, the game automatically changes to add a bit of fun by revising the look of several game elements.

Participating

Upon loading of game save, the april fools' day event will be active unless it was disabled beforehand.

Disabling the event

The event can be disabled using the "Disable Seasonal Events" option. This option is only available on the title screen , under Options > Gameplay > Disable Seasonal Events .

In multiplayer , the host's configuration is used. If the host has seasonal events enabled, but the client does not, the event will be enabled in the session.

Alternatively, seasonal events can be disabled using the -DisableSeasonalEvents launch argument . On a dedicated server , this launch argument is the only way to disable seasonal events.

Changes made

Pioneer

The Pioneer helmets appeared larger.

  • Pioneer with Big Head / Helmet.
  • A group of Pioneer "Clones" with Big Heads / Helmets.
  • Big Head Pioneer Travel Through Hyper Tube.

Lizard Doggo

All Lizard Doggo's appeared larger, with some of them also wearing the Pioneer's large helmet.

  • Group of Large Lizard Doggo's
  • Large Lizard Doggo with Pioneer Helmet

Space Giraffe-Tick-Penguin-Whale Thing

All Space Giraffe-Tick-Penguin-Whale Thing's appeared smaller.

  • Smaller Space Giraffe-Tick-Penguin-Whale-Thing and Pioneer with Big Head / Helmet
  • Smaller Space Giraffe-Tick-Penguin-Whale-Thing
  • Smaller Space Giraffe-Tick-Penguin-Whale-Thing

Fluffy-tailed Hog

All Fluffy-tailed Hogs were changed to look like the Walker , an enemy is from Sanctum 2 , another game by Coffee Stain Studios .

  • Fluffy-tail Hogs appearing as Walkers

Cliff Hog and Nuclear Hog

Starting with the April Fools Event 2024, all Cliff Hogs and Nuclear Hogs were changed to have a shape similar as the Walker , but still used their default skin appearance.

  • Cliff Hogs appearing in the shape of Walkers
  • Nuclear Hogs appearing in the shape of Walkers

Resource Scanner

There were random capitalization of resource names in the Resource Scanner, except for Coal .

  • Resource Scanner showing scan for Sulfur with randomized capitalization of Resource Name

Flora

Some Flora was replaced with random equipment , which made it appear in strange sizes, and sometimes in mid-air
