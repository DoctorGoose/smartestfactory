  • Module:Achievement

This template is used to display achievements within the table of all achievements. Achievements are taken from Template:Achievements.json .

Usage

This template creates a whole table row, and must be inserted into a table.

Parameters

Parameters have to be entered in order.

Parameter Data type Description
1 String Internal name of the achievement (required)
2 String Notes to display in the last column (optional)
