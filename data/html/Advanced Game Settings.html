Advanced Game Settings ( AGS ) [ 1 ] are essentially cheats that allow customizing the Satisfactory experience.

Enabling

Advanced Game Settings can be enabled in two ways: [ 2 ]

  • When starting a new session: Clicking the 'Advanced Game Settings' button.
  • When loading an existing save file: Checking the 'Enable Advanced Game Settings' box below the 'Load Game' button in the 'Load' menu. Note that this checkbox is not present for saves that already have AGS enabled.

In both cases the player must confirm a warning prompt before they are allowed to proceed, acknowledging that they understand the effects of AGS on their save file, which includes disabling achievements .

CAUTION: It is not possible to disable AGS on a particular save file once they have been enabled.

Once enabled, individual settings can be toggled on and off from the Esc menu in-game . In multiplayer , all players have access to the settings, not only the host. Select settings affect only the player who enables them (such as Flight), while others affect all players (such as No Unlock Cost).

AGS Game Saves

Individual game saves with AGS enables will be shown in the Load Game Save Menu list with a ✎ Pencil Icon. [ 3 ]

Multiplayer / Dedicated Server

The Multiplayer host, or Dedicated Server admin can enable AGS settings. Clients will have access to all AGS settings on an AGS enable game save. [ 4 ]

Certain AGS settings apply to an individual only or apply to everyone. [ 5 ]

List of AGS

Gameplay

Setting Affects Effects
No Build Cost Only you Building and customizing does not cost any resources, dismantling does not return any resources [ 6 ]
Flight Mode Only you Unlimited flight by double-tapping Space (See AGS Flight Mode Controls ) [ 7 ]
No Power Everyone Buildings don't require any Power to run, not even Power Line connections [ 8 ]
No Unlock Cost Everyone Milestones , MAM research , AWESOME Shop purchases and Project Assembly phases don't require anything to unlock [ 9 ]
Unlock Alternative Recipes Instantly Everyone Alternate Recipes will be unlocked immediately upon unlocking the condition for them to otherwise be unlockable from Hard Drives (denoted as "Unlocked by" on the wiki) [ 10 ]
Keep Inventory Only you Determines what happens to inventory upon death. Keep equipment only (Default), or keep everything, or keep nothing [ 11 ]
Give Items Only you Gives selected items or item stacks to the inventory (if there is space) [ 12 ]
God Mode Only you Grants invincibility from every damage source except the World border which will still kill you [ 13 ]
No Fuel Only you Vehicles no longer require fuel to run.

Creatures

Setting Affects Effects
Disable Arachnid Creatures Everyone Stingers do not spawn at all [ 14 ]

Progression

Setting Affects Effects
Set Starting Tier Everyone Sets specific starting Tier to begin or unlocks all Tiers. Can only be done when starting a new Game [ 15 ]
Set Game Phase Everyone Sets the Project Assembly phase [ 16 ]
Unlock All Tiers Everyone Unlocks all Milestones [ 17 ]
Unlock All Research in the MAM Everyone Unlocks all MAM research nodes [ 17 ]
Unlock All in the AWESOME Shop Everyone Unlocks all permanent AWESOME Shop purchases (building and customizer schematics) [ 17 ]

History

  • Patch 1.1.0.0 : The highly requested "No fuel" option has finally been added to Advanced Game Settings, so you if you enjoy playing with AGS, you no longer have to do extra steps when using vehicles
  • Patch 1.0.0.5 : Fixed Advanced Game Settings being activated unintentionally transferring from the current save to another when starting a new session through the Server Manager (on Dedicated servers )
  • Patch 1.0.0.4 : Fixed items still refunding materials and clogging up the Inventory when dismantling and using No Build Cost in Advanced Game Settings
  • Patch 0.8.2.1 : Made it so offline Players can now take damage regardless of any other conditions like God Mode enabled on Advanced Game Settings
  • Patch 0.8.2.0
    • Fixed Multiplayer / Dedicated Servers Clients not continuing to have Flight Mode enabled after logging in and out of a session
    • Removed knockback when the player has Flight Mode enabled
    • Flight Mode is no longer turned off when you use Turbo Bass with the Boombox while it is enabled
    • Fixed Game Phase Numbers and Naming in the Advanced Game Settings menu
  • Patch 0.8.0.0 : Introduced
