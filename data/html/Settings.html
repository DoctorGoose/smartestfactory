Options

There are several sub-tabs in the 'Options' tab in the main menu.

Gameplay

Gameplay

Setting Options Default Effects
Creature Hostility Default Adjust the hostility settings for all creatures in the world.
Default Creatures will attack players on sight
Passive Creatures will not attack players, or retaliate when attacked by players
Retaliate Creatures will only attack when attacked by players
Keep Inventory Keep Nothing, Keep Equipment, Keep Everything Keep Equipment Affects what items are kept on death
Automatic Clearance Separation On/Off Off When enabled, buildings are automatically moved if they are built within another buildings clearance
Autosave Interval (minutes) 0-120 5 Sets the amount of time in minutes for the autosave interval.
0 disables autosaves entirely. The amount of autosaves cannot be changed and is always 3
Dismantle Crate Placement Next to Player, On Dismantle Site Next to Player Determines the placement of the Dismantle Crate when dismantling buildings
Merge Dismantle Crates On/Off On When enabled, dismantling will try to put resources into the existing dismantle crate in 100m range before making a new dismantle crate
Sample Copy Customization On/Off On When enabled, will copy customization on a building when sampling it
(see Building eyedropper )
Sample Copy Settings On/Off On When enabled, will copy settings such as recipe and overclocking on a building when sampling it
(see Building eyedropper )
Server Restart Schedule (time) 00:00 to 24:00 24:00 Sets the time of day the server should restart at. So if it is set to 17:00, the server will restart at 17:00 every day
(uses 24-hour clock)
Show Clearance On/Off On Enables or disables showing the white hitbox representing the hologram's clearance when building
Arachnophobia Mode On/Off Off When enabled, Stingers (spider-like creatures) are replaced with 2D cat sprites. This replaces Stinger sounds with meows
This setting helps make the game more palatable to those with arachnophobia .
Enable Mouse Smoothing On/Off On When enabled, makes mouse movements smoother. Essentially, this delays your cursor movements to create a smoother feeling
Disable Narrative Messages On/Off Off Enables or disables the narrative messages from ADA from showing up onscreen for you. It does not prevent other players from seeing messages.
The muted messages will still appear in the Codex , where you can play them at will
Send Gameplay Data On/Off On Unchecking this box means you will not send data to Coffee Stain Studios when playing
(changing this value requires you to restart game for the change to take effect)
Network Quality Low, Medium, High, Ultra Medium Increasing network quality may improve client load times and network performance at the cost of server frame-rate. Experiment with these settings to see what works for you.
(see Multiplayer Temporary Lag Solution for more details)
Disable Seasonal Events On/Off Off Removes all seasonal event content from the game, such as FICSMAS

Performance & Graphics Quality

Setting Options Default Effects
Texture Quality Low, Medium, High, Ultra Ultra Sets the quality of object textures in the game
Shadow Quality Sets the quality of shadows in the game
Far Shadow Trace Distance Near, Default, Far, Cinematic Default Distance that far shadows will trace, lower values can cause gaps in the shadows. High values will cost more performance on the GPU
Post Processing Low, Medium, High, Ultra Ultra Affects post processing effects, such as motion blur, fog, etc.
Anti Aliasing Turn on to prevent jagged lines along edges.
The "Low" setting disables it entirely.
Foliage Load Distance Near, Default, Far, Very Far, Cinematic Far Controls how quickly foliage will load in
Use the "Cinematic" setting at your own risk.
VFX Quality Low, Medium, High, Ultra Ultra Affects the quality of visual effects in the game, such as smoke
Foliage Quality Sets the quality of foliage/vegetation in the game
Light Quality A higher setting means more lights will be rendered onscreen at once, and a lower setting means fewer lights will be rendered at once
(see also Lights )
View Distance Sets when models in the distance will start to load in LODs which are lower quality models that are loaded in to increase performance
Cloud Quality Sets the quality of clouds in the game
Motion Blur On/Off Off Enable to see motion blur when moving the camera in the game
Conveyor Visual Quality Low, Medium, High, Ultra Medium Increased the visual accuracy of the items on conveyor belts for a minor CPU and memory cost
(does not affect gameplay)

Setting Options Default Effects
Upscaling Method None, Nvidia DLSS, Intel XeSS, Temporal Super Resolution (TSR), AMD FidelityFX™ Super Resolution (FSR) Temporal Super Resolution (TSR) Select Upscaling Method

Anti-Aliasing Method None, Fast approximate AA (FXAA), Temporal AA (TAA), Temporal Super Resolution (TSR) Temporal Super Resolution (TSR) Select the desired Anti-Aliasing method

Frame Generation On/Off Off Uses AI to create additional frames between rendered ones, boosting frame rates and smoothing motion
Upscale Preset Performance, Balanced, Quality, No Downscaling Balanced Sets the preferred upscaling to use

Balanced (balance between performance and quality)
Quality (quality over performance)

Screen Percentage 0-100 75 The percentage of the resolution the game will be rendered at
Example: 50% means the game will be rendered at half the resolution, but scaled up to fit the entire window size.

Conveyor Belt Item Frequency Capped at 30, Capped at 60, Unlimited Unlimited Sets how frequent the visuals of items on conveyor belts are updated

Conveyor Belt Render Distance Near, Medium, Far, Ultra Far The distance at which items on conveyor belts will be rendered
Gamma 0-3 (increments of 0.05) 2 Controls the overall brightness in the game
(see also Gamma for more information)
VSync On/Off Off Enables vertical synchronization. This can prevent screen tearing
Hierarchical Z-Buffer Occlusion On/Off Off Lowers the quality of the depth buffer. May have a performance impact on some systems
(reduces CPU use and increases GPU use for rendering)
Enable LOD Dithering On/Off On Enables smoother transitioning between lower-detail models
Enable Contact Shadows On/Off On Enables higher quality shadows
Global Illumination Off, Medium, High Off Enables Lumen for Global Illumination. This can be expanded using a small arrow ◀ to the right, where the lighting mood can be adjusted.
Lighting Mood Dark, Moody Dark, Moody Bright, Bright Bright Changes the mood in enclosed spaces
(only visible if Global Illumination is expanded using a small arrow ◀ to the right)

General

Setting Options Default Effects
Select Controls Keyboard and Mouse, Controller Keyboard and Mouse Select if you want to play with Keyboard and Mouse or Controller
Unavailable if a controller is not detected
Controller Type Default, Xbox Series X/S, DuelSense Default Select your desired controller type to be displayed
Unavailable if Select Controls is not set to Controller
Invert Look On/Off Off Inverts the X and Y axis when controlling the camera (for looking around with the mouse)
Mouse Sensitivity 1-10 4 Affects the mouse sensitivity
Hold To Snap Building On/Off On If enabled, hold Left Ctrl to snap buildings to the grid
(controller only)
Hold To Sprint On/Off On When enabled, ⇧ Left Shift to keep sprinting. When disabled, pressing ⇧ Left Shift once will cause you to continue sprinting until you press ⇧ Left Shift again
Hold To Zipline On/Off On If enabled, aim at Power Line and hold the button to zip along the line. If disabled, aim at the Power Line and press the button once to start and press it a second time to stop. Eliminates the need to hold the button when traveling.
Controlled X Axis Rotation On/Off On When enabled, Left Ctrl and aim to set the vertical rotation of structures with X axis rotation, such as Pipelines and Hypertubes . Aiming without holding the input will set the structure's support height.
When disabled, these controls are inverted: hold Left Ctrl and aim to set the support height, and set vertical rotation by aiming without holding the input.
Equipment Swapping Scroll Direction Right/Left Left The direction of (Scroll Up/Down) when cycling through hand slot equipment . "Left" means scrolling down will cycle to the left. "Right" means scrolling down will cycle to the right
Hologram Rotation Mode Static, Spawn Facing Player, Always Face Player Static Controls how hologram rotation is applied relative to player's location. Hologram Rotation Mode can be expanded using a small arrow ◀ to the right, where how the hologram is rotated relative to the player.
Relative Rotation to Player Perpendicular, Parallel Perpendicular Controls how the hologram is rotated relative to the player. Available options are perpendicular (side facing the player) and parallel (front facing the player)
(Only visible if Hologram Rotation Mode is expanded using a small arrow ◀ to the right)
Hotbar Scroll Direction Clockwise/Counterclockwise Clockwise Direction of Right Alt + (Scroll Up/Down) when cycling through Hotbars . Clockwise means scrolling up will cycle clockwise (1 to 10), and counterclockwise means scrolling up will cycle counterclockwise (10 to 1)
One-Click Build On/Off Off Buildings placed with two clicks, such as Conveyor Belts with Conveyor Poles attached, will now be placed with one click. Hold to change the height of the building, then release to place.
