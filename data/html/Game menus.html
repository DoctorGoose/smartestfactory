Satisfactory utilizes several types of menus , most notably the main menu (title screen) and escape menu.

Some other examples include the Inventory ( Tab ↹ ), radial menus such as the Resource Scanner , building configuration menus, etc.

Main menu

The main menu is an introductory interface seen when the game starts. Prominently featuring the Satisfactory logo and a thematic background, it allows a game session to start by creating a new one or loading a save file . It also gives access to the settings menu.

Menu options

Button Usage Notes
Continue Loads the most recent save file, starting a game session. Only available if any save file exists
New Game Brings up the New Game Start Screen , allowing a new session to be started. Starts new game session. Both Multiplayer Settings and Advanced Game Settings can be enabled before starting.
Join Game Allows the Multiplayer Clients to join another Players current Multiplayer Session. See the linked page for details. See also Multiplayer
Load Loads the selected save file, starting a game session. Advanced Game Settings can be enabled before loading.
Save files can also be deleted in this menu.
Options Allows access to the settings menu. See also Settings
Online Settings Allows accepting invites from other players. See also Multiplayer .
Server Manager Allows the management of Dedicated servers. See also Dedicated servers
Mods Allows access to Mod configuration files. ⭑ Menu option only appears if Mods are loaded.
Credits Shows game credits.
Exit Closes the game and exits to desktop.

Current game info

Below the menu options the following current game information overlay is displayed:

  • Player Account and Name - Shows the Player Account (Steam or Epic) and Player Account Name.
  • Version - Shows current build number .

Game mods loaded

In addition to the current game information, the following will also be shown if any game mods are being used:

  • Mod Loader - Shows current version of the Satisfactory Mod Manager (SMM) and Satisfactory Mod Loader (SML) used.
    • The SML is a game mod that is automatically added and updated via the SMM.
  • Mods - Shows the number of game mods that are currently loaded.

Patch notes

To the right on the Main Menu the latest Patch Notes will be shown in abbreviated form. Click Read More will show the full version of latest patch notes.

Social media

In the lower right, below patch notes, there will be icons (official game related links) that open in a browser window for the following (in the order shown) :

  • Coffee Stain Studio Youtube Channel
  • Satisfactory Wiki ⭑ (you are here)

Offline warning

Starting an offline session will show a popup to inform them that an online session could not be created. You can retry the sign-in process by restarting the game.

Escape menu

The in-game escape menu, as the name suggests, is shown by pressing Esc .

Menu options

Button Usage Notes
Resume Game Closes the menu.
Save Game Allows saving the current session by creating a save file.
Load Game Allows loading a different save file, starting a previous game from current session, or starting a different game session. Advanced Game Settings can be enabled before loading one.
Manage Session Shows a list of currently connected players, allows changing the session privacy, etc. See also Multiplayer
Go to feedback site Opens the Q&A site in the default web browser.
Options Allows access to the settings menu. See also Settings
Server Manager Allows the management of Dedicated servers. See also Dedicated servers
Mods Allows access to Mod configuration files. ⭑ Menu option only appears if Mods are loaded.
Mod Savegame Settings Current Mod configuration files settings. ⭑ Menu option only appears if Mods are loaded.
Exit to Main Menu Exits to the main menu . The game is not saved automatically
Exit to Desktop Closes game and exits to desktop. The game is not saved automatically
Respawn Instantly kills your character. This allows you to respawn if stuck. There is no confirmation prompt. Other players in multiplayer can still revive you.

Current game info

Below the menu options the following current game information overlay is displayed:

  • Current Session - Shows name of the Game Save that is currently loaded.
  • Player Account and Name - Shows the Player Account (Steam or Epic) and Player Account Name.
  • Version - Shows current build number .

Game mods loaded

In addition to the current game information, the following will also be shown if any game mods are being used:

  • Mod Loader - Shows current version of the Satisfactory Mod Manager (SMM) and Satisfactory Mod Loader (SML) used.
    • The SML is a game mod that is automatically added and updated via the SMM.
  • Mods - Shows the number of game mods that are currently loaded.
