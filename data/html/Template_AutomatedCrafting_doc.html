This template uses Lua :
  • Module:RenderCraftingRecipesTable

Invokes Module:RenderCraftingRecipesTable , which generates a table containing all crafting recipes for specified item and items/minute calculations.
Data for the table is taken from ' crafting_recipes ' cargo table.

Usage

{{RenderCraftingTables|Reinforced Iron Plate}}

You can omit item name, in which case {{PAGENAME}} value will be used automatically: {{RenderCraftingTables}} (on Reinforced Iron Plate page).

{{RenderCraftingTables|Reinforced Iron Plate|experimental}} will display only recipes that are available only on Experimental build of the game.

Lua error: Error: Table crafting_recipes not found..
