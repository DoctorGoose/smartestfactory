Conveyor Poles are logistic structures used to support Conveyor Belts and Conveyor Lifts .

All kinds of Poles allow a Conveyor to be attached to it from both sides, as long as the Conveyor is set to transport in the same direction, which can be especially useful to force the direction of Conveyor Lifts. Conveyors keep on working even if the Poles supporting them are dismantled.

Conveyor Lifts can be built standalone without supports, unlike Belts.

Construction

Standard Conveyor Poles are built by clicking ( ) once to set the location, then scrolling to rotate and looking up or down to change the height (if desired), confirming the placement by clicking again.

Stackable Conveyor Poles are built simply by placing them on the ground, on each other or other kinds of stackable supports. It's not possible to make them the default type of Poles used when building Conveyor Belts. Simple Conveyor Poles fit under Stackable Poles. Stackable Conveyor Poles can be built one at a time or 10 at once using the Zoop construction mode.

Conveyor Wall Mounts can only be built on the sides of Walls and Foundations, as mentioned above. It's not possible to place them on ceilings, unlike Pipeline or Hyper Tube Wall Supports . For that purpose, Conveyor Ceiling Mounts can be used.

Conveyor Ceiling Mounts now allow for granular rotation when holding the Snap To Guidelines Left Ctrl Key.

⭑ NOTE: If the Build In One Click option is enabled, has to be held down to adjust the rotation and height while placing.
