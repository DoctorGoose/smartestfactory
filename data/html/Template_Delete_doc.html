[ purge ]

This template will add tagged pages to Category:Candidates for deletion or Category:Candidates for speedy deletion .

Note: Put the tag between <noinclude> </noinclude> tags for deletion of transcluded pages.

Usage

{{ delete |Regular deletion reason}}

{{ delete |Speedy deletion reason|speedy}}

Parameters

parameter 1
reason for proposed deletion
parameter 2
Optional. For speedy deletions: speedy
