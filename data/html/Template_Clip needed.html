• Tells a visitor that the statement is not yet supported by a reference to a clip from a livestream VOD or video
  • Displays [ Clip needed ] when given no description parameter
  • Displays [ Clip needed ] when given a description parameter
  • Editors are encouraged to use the description parameter (e.g. {{clip needed|DESCRIPTION GOES HERE}}) to indicate where a clip may be sourced from.
  • If no clip is known, vaguer date ranges should be used but too much uncertainty should require {{ citation needed }} be used instead.

Examples

  • {{clip needed}} adds [ Clip needed ]
  • {{clip needed|August 31st, 2021 Livestream}} adds [ Clip needed ]
