Somersloop

An odd alien object with a mind-bending, yet somehow familiar shape. It emits an unsettling energy.

Class name

Desc_WAT1_C

Item

Stack size

50

索莫晶体 是在 世界 各地发现的外星人工制品，用于 生产增幅 和制造 外星能源强化装置 。

它们会发出红光和不寻常的粒子，还会发出时大时小的脉动声。长时间站在它们旁边会听到声音，敦促玩家收集它们。

获取

在探索过程中可以发现索莫晶体，通常位于难以到达的地方，这些地方通常有敌人把守，周围有 铀矿藏 、 孢子花 、 破裂的巨石 或 毒气柱 。 1.0版本 总共有106个索莫晶体。

用法

生产增幅

主要文章: 生产增幅

索莫晶体用于 生产增幅 。在选定的生产建筑中插入索莫晶体，可以在不增加投入成本的情况下提高建筑的产出。最大提升幅度为 +100%，总产出翻倍。

琐事

  • 索莫晶体引用了“ Cool S ”，这是一个流行的符号，通常被涂鸦在儿童笔记本上。它被描述为 “某种熟悉的形状”，这可能反映出 S 图的起源和传播并不明确。
