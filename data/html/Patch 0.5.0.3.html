Patch Notes: Early Access (EXPERIMENTAL) – v0.5.0.3 – Build 170429. This patch was released on 27 October 2021 . It only fixes a crash from Patch 0.5.0.2 and is otherwise identical.

Intro

Hi Pioneers!

As you may or may not have noticed, funnily enough we did introduce some new issues™️ on the previous build, there was an EOS related crash when starting a new game or loading a save which has now been fixed.

So you can now enjoy the previous fix to the fullest :D

Bug fixes

  • Fixed Epic Online Services (EOS) related crash when loading or starting a new game

To avoid people missing what the previous patch contained, I’m including it alongside this post in it’s original formatting (see Patch 0.5.0.2 ).

Sorry for the inconvenience and confusion there but it’s all been sorted out.

Until next time!
