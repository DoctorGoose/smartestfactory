The Ping Function has been in the game since pre-Closed Alpha [ 1 ] with the notable change from using an "X" character for the "Ping" to using an exclamation mark "!" character [ 2 ] for the "Ping" at the Ping location.

The player can place a Ping at the location where the cursor is by pressing Alt + Left Mouse Button . The ping indicator shows up as an exclamation mark and is now visible on the on-screen Compass and Map . The ping also makes a sound coming from the ping location. Pings can be used to point at things for other players to see.

Current issues

  • Sometimes the player can get stuck in ping mode after using Alt + Tab ↹ to switch windows out of the game, resulting in every click being interpreted as a ping regardless of if Alt is pressed. Press Alt again to resolve this, returning clicking functionality to normal.

Trivia

Ping can be used to survey terrain distance, although it has a limited range ~500 metres.

History
