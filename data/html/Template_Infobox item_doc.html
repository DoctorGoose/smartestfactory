  • Module:RenderInfobox

Infobox item is an automatically generated infobox that sources its data from Docs.json. Its building counterpart is {{ Infobox building }} .

Parameters

The className argument is the only mandatory argument. A number of equipment-related parameters can also be supplied, as they cannot be acquired from Docs.json.

Parameter Usage
className className of the item (mandatory)
ammo Accepted ammo (list of {{ ItemLink }} )
damage Damage per hit
magSize Magazine size per reload
rateOfFire Rate of fire (s)
reloadTime Reload time (s)
damagePerSecond Damage per second
range Range (does not include unit)
