The HistoryMissing template is used to mark pages on which the history section is missing completely or is incomplete. Pages with this template are automatically placed into Category:Missing history .

Parameters

Parameter Data Type Description
1 String If set to "incomplete", changes the template to say that the history section is incomplete rather than missing completely, any other value is ignored
